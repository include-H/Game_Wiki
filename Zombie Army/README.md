# 《僵尸部队》系列 Wiki 目录

> 说明：本目录用于集中整理《僵尸部队》（Zombie Army）系列相关 Wiki 条目。这个系列最早以《Sniper Elite: Nazi Zombie Army》的名义，从《狙击精英 V2》的技术与玩法框架里分化出来，后来逐步摆脱“狙击精英丧尸外传”的身份，发展成一条以四人合作、尸潮压迫、恶趣味恐怖和夸张肢解演出为主的独立 Rebellion 射击系列。

---

## 1. 系列概览

《僵尸部队》最特别的地方，在于它并不是简单把《狙击精英》的敌人换成僵尸。早期两作确实直接沿用了《Sniper Elite V2》的骨架：柏林废墟、Karl Fairburne、X-Ray Kill Cam 和第三人称射击手感都还在。但从设计目标看，它已经明显转向另一条路：更强调四人合作、波次守点、压迫感很强的线性推进，以及“越到后面越像地狱大片”的怪物升级。

如果按系列发展来看，这条线大致可以分成两段：

- `Nazi Zombie Army` 阶段：2013 年的《Sniper Elite: Nazi Zombie Army》与《Sniper Elite: Nazi Zombie Army 2》，仍然保留《狙击精英》前缀，像一组从 `V2` 里长出来的独立丧尸扩展作。
- `Zombie Army` 独立阶段：从 2015 年《Zombie Army Trilogy》开始，系列正式摆脱《Sniper Elite》前缀，改以“僵尸部队”独立品牌存在；到《Zombie Army 4: Dead War》时，这条线已经完全确立为合作尸潮射击系列。

需要额外说明的是，《Zombie Army VR》属于后续独立旁支，时间线上承接主系列，但游玩形态和主角群组织方式都有明显区别，更适合后续单独展开。

## 3. 阅读顺序

如果是想按系列内部演化来读，最自然的顺序是：

1. 《Sniper Elite: Nazi Zombie Army》
2. 《Sniper Elite: Nazi Zombie Army 2》
3. 《Zombie Army Trilogy》
4. 《Zombie Army 4: Dead War》
5. 《Zombie Army VR》

如果只是想快速理解“《僵尸部队》为什么会从外传长成独立系列”，也可以先读《Zombie Army Trilogy》，再回头看前两作。因为 Trilogy 同时承担了三件很重要的事：

- 把前两作重新整合进统一入口
- 补上原本不存在的第三章
- 让系列正式从《Sniper Elite》前缀中独立出来

## 5. 版本关系与补充说明

- 《Sniper Elite: Nazi Zombie Army》与《Sniper Elite: Nazi Zombie Army 2》都是独立运行的作品，不需要《Sniper Elite V2》本体才能游玩；但它们在技术、手感和世界观血缘上都明显来自 `V2`。
- 《Zombie Army Trilogy》不是单纯把旧作打包出售，而是“复刻 1 代 + 复刻 2 代 + 新增第 3 章”的完整重组版本；对多数今天的玩家来说，它也是理解早期《僵尸部队》的主要入口。
- 《Zombie Army 4: Dead War》是对 Trilogy 的正式续作，不再局限于柏林收尾，而是把“Dead War”扩展成横跨意大利与更广区域的持续战争。
- 《Zombie Army 4》后续推出了多季 `Season Pass` 和 `Ragnarök` 战役扩展，对完整体验和剧情收束都有明显意义，因此在正文里需要一并说明。
- 《Zombie Army VR》承接主系列世界观，但更适合视为 VR 旁支，而不是《Zombie Army 5》这类主线编号续作。

---

## 6. 一句话总结

《僵尸部队》系列真正有趣的地方，是它从《狙击精英》的尸潮外传起步，最后却靠合作推进、恶魔纳粹和夸张肢解演出，长成了一条风格越来越独立的 Rebellion 招牌副线。
