## 1. 系列概览

《仙剑奇侠传》是中国游戏史上最具影响力的国产 RPG 系列，也是华语单机游戏中最具文化符号意义的品牌之一。自 1995 年初代问世以来，它以"仙侠世界观 + 宿命悲剧 + 情感叙事"为核心，在二十余年间塑造了李逍遥、赵灵儿、林月如、景天、云天河、姜云凡等一批深入人心的角色，也留下了"十年一剑情如梦""莫失莫忘"等经典意象。

和很多长期连载系列不同，《仙剑奇侠传》的核心魅力并不在于玩法机制的持续进化，而在于它始终把"情感叙事"放在最中心的位置。从初代的"仙灵岛上觅仙踪"到四代的"我命由我不由天"，再到五前传的"羁旅天涯"，每一作都在用不同的方式回答同一个问题：在宿命与选择之间，人究竟能走多远。

如果按系列内部的开发团队与风格演变来看，《仙剑奇侠传》大致可以分成四个阶段：

- **经典奠基阶段**：初代《仙剑奇侠传》（1995）及其重制版本
- **台北总部延续阶段**：《仙剑奇侠传二》（2003）
- **上海软星黄金阶段**：《仙剑奇侠传三》（2003）、《仙剑奇侠传三外传·问情篇》（2004）、《仙剑奇侠传四》（2007）
- **北京软星接力阶段**：《仙剑奇侠传五》（2011）、《仙剑奇侠传五前传》（2013）、《仙剑奇侠传六》（2015）、《仙剑奇侠传七》（2021）

这条系列最有意思的地方，在于它既是一条游戏产品线，也是华语流行文化中少有的"跨世代情感记忆载体"。很多玩家是从 DOS 时代一路追到今天，也有人是通过电视剧、同人作品或口碑传播反向入坑。无论哪种方式，《仙剑奇侠传》都已经不只是"一款游戏"，而是一种文化现象。

## 2. 分期方式

### 2.1 经典奠基阶段

- [《仙剑奇侠传》（1995）Wiki](./PAL_1995_WIKI.md)

这一阶段决定了系列最核心的气质。初代以 DOS 平台起步，后经 Windows 移植、重制与多次再发行，奠定了"仙侠世界观 + 宿命悲剧 + 情感叙事"的基本公式。李逍遥、赵灵儿、林月如的三角关系，以及"仙灵岛""锁妖塔""十年之约"等经典桥段，至今仍是整个系列最核心的文化符号。

### 2.2 台北总部延续阶段

- [《仙剑奇侠传二》Wiki](./PAL_2_WIKI.md)

这一作是系列正传续作的开端，由台北总部原班人马主导开发。它在剧情上承接初代，以王小虎为主角，试图延续并扩展仙剑世界观。但由于开发周期、团队变动与玩家期待落差等多重因素，本作在系列史上的评价相对复杂。

### 2.3 上海软星黄金阶段

- [《仙剑奇侠传三》Wiki](./PAL_3_WIKI.md)
- [《仙剑奇侠传三外传·问情篇》Wiki](./PAL_3A_WENQING_WIKI.md)
- [《仙剑奇侠传四》Wiki](./PAL_4_WIKI.md)

这一阶段是《仙剑奇侠传》系列口碑与影响力的又一高峰。上海软星团队在保持情感叙事核心的同时，引入了更成熟的世界观架构、更丰富的角色塑造和更完善的玩法系统。《仙剑三》的"六界设定"与"轮回主题"、《问情篇》的"蜀山前史"、《仙剑四》的"我命由我不由天"，共同构成了这一阶段的独特气质。

### 2.4 北京软星接力阶段

- [《仙剑奇侠传五》Wiki](./PAL_5_WIKI.md)
- [《仙剑奇侠传五前传》Wiki](./PAL_5A_PREQUEL_WIKI.md)
- [《仙剑奇侠传六》Wiki](./PAL_6_WIKI.md)
- [《仙剑奇侠传七》Wiki](./PAL_7_WIKI.md)

上海软星解散后，北京软星接过系列开发接力棒。这一阶段的作品在技术层面有明显提升，同时也面临着玩家期待、市场环境与开发资源的多重挑战。《仙剑五》与《仙剑六》的评价相对分化，而《五前传》则被许多玩家视为这一阶段的口碑高点。《仙剑七》作为系列最新正传，在画面与战斗系统上进行了全面革新。

## 3. 阅读顺序

如果想按剧情时间线理解系列世界观，更稳妥的读法是：

1. 《仙剑奇侠传四》：时间位置较早，但与主线人物谱系相对独立
2. 《仙剑奇侠传三》
3. 《仙剑奇侠传三外传·问情篇》
4. 《仙剑奇侠传》（1995）
5. 《仙剑奇侠传二》
6. 《仙剑奇侠传五前传》
7. 《仙剑奇侠传五》
8. 《仙剑奇侠传六》：相对独立，年代约在五前传之后很久
9. 《仙剑奇侠传七》：与初代、二代、三代、三外、五代、五前传属于连贯世界线，但时代更靠后

如果想按发售顺序理解系列演化，推荐这样读：

1. 《仙剑奇侠传》（1995）
2. 《仙剑奇侠传二》
3. 《仙剑奇侠传三》
4. 《仙剑奇侠传三外传·问情篇》
5. 《仙剑奇侠传四》
6. 《仙剑奇侠传五》
7. 《仙剑奇侠传五前传》
8. 《仙剑奇侠传六》
9. 《仙剑奇侠传七》

如果只是想先理解《仙剑奇侠传》为什么能在华语游戏史上占住位置，当前最适合先读这三作：

1. 《仙剑奇侠传》（1995）—— 系列原点，理解一切的基础
2. 《仙剑奇侠传四》—— 上海软星时期的口碑巅峰
3. 《仙剑奇侠传五前传》—— 北京软星时期的最佳表现

## 5. 版本关系与补充说明

《仙剑奇侠传》系列最容易混淆的地方，主要集中在"初代多版本""外传定位"和"开发团队更迭"这几类关系上。

### 5.1 初代版本关系

初代《仙剑奇侠传》存在多个重要版本，需要特别注意区分：

- **DOS 版（1995）**：系列原点，最初发行版本
- **Windows 95 版 / 98 柔情篇（1997）**：Windows 移植版，画面与音乐有所优化，是最广泛传播的版本
- **新仙剑奇侠传（2001）**：完全重制版，画面全面升级，剧情有少量扩展
- **新仙剑奇侠传电视剧纪念 XP 版（2005）**：配合电视剧发行的重制版本
- **仙剑奇侠传 Mobile / Steam 版**：后续数字发行版本

讨论初代时，若不特别说明，默认以 98 柔情篇为基准版本。

### 5.2 外传定位

系列中的"外传"作品并非简单的衍生作，而是与正传世界观紧密关联的重要作品：

- 《仙剑奇侠传三外传·问情篇》：以蜀山派为核心的前传性质作品，补完仙剑三世界观
- 《仙剑奇侠传五前传》：以五代前史为核心，被视为北京软星时期口碑最佳的作品

这两作在剧情完整性和角色塑造上都有很高评价，不应被简单视为"配角作品"。

### 5.3 开发团队更迭

系列开发团队经历了多次重大变动：

- **狂徒创作群**：初代（1995），并参与《仙剑奇侠传二》前期开发
- **上海软星**：三代、三外、四代（2007 年解散）
- **北京软星**：五代、五前、六代、七代

这一变动直接影响了各阶段的风格差异，也是理解系列评价分化的重要背景。

### 5.4 世界观关联

系列世界观存在复杂的关联关系：

- 初代、二代、三代、三外、五代、五前传、七代属于连贯世界线
- 四代与六代都相对独立，但仍共享系列总体神话框架与部分共通设定

## 7. 一句话总结

《仙剑奇侠传》之所以能成为华语游戏史上的文化符号，不只是因为它活得够久，而是因为它在二十余年间，始终在用不同的方式回答同一个问题：在宿命与选择之间，人究竟能走多远。
