# 游戏 Wiki 总目录

> 说明：本目录用于集中整理当前项目中的游戏 Wiki 条目、系列索引与专题入口。整体以“中文整理型游戏 Wiki”为目标，强调结构清晰、资料可靠、便于快速检索。当前目录已逐步形成“综合游戏系列 + Galgame 专区 + 时代 / 杂项专题”三层结构。

---

## 1. 目录结构

当前 `Wiki` 目录大致可以按下面三块理解：

- 综合游戏系列：动作、射击、RPG、策略、竞速等主机 / PC 游戏系列主干
- `Gal` 专区：Galgame / 视觉小说相关条目与系列目录
- 时代与杂项专题：如“辉煌时代”“未有系列”这类更适合专题化整理的入口

这样拆分之后，`Wiki/README.md` 不再负责把每一部作品都拉平罗列，而是优先提供清楚的一级导航。真正细到“这一系列当前已写到哪一作”，应优先进入对应目录的 `README.md` 查看。

## 5. 当前整理思路

现在这份总目录遵循几条比较明确的原则：

- 一级目录优先：`Wiki/README.md` 只负责总导航，不再承担过细的“单作覆盖清单”
- 系列内细节下放：具体到某个系列当前写到哪些作品，进入对应目录 `README.md` 查看
- `Gal` 独立成区：视觉小说 / Galgame 不再分散挂在总目录各处，而是集中到 [《Gal》Wiki 目录](./Gal/README.md)
- 便于继续扩展：未来新建系列、补系列条目或新增单作专题时，只需要在总目录补一级入口，不必频繁重写整份 README

这样做的好处是，总目录会更稳定，也更适合作为真正的“入口页”。

---

## 6. 推荐使用方式

如果是第一次进入这个项目，比较推荐按下面方式浏览：

1. 想找某个知名系列：直接从上面的综合系列目录进入
2. 想看视觉小说 / Galgame：直接进入 [《Gal》Wiki 目录](./Gal/README.md)
3. 想看时代、平台或尚未完全成系列的专题：看 [《辉煌时代》专题](./辉煌时代/README.md) 与 [《未有系列》专题](./未有系列/README.md)

换句话说，这份 README 现在更像一张“馆内地图”，而不是作品总表。

---

## 7. 一句话总结

这份 `Wiki` 总目录真正想做的，不是把所有条目平铺出来，而是让每一类作品、每一条系列和每一个专题，都有一个清楚、稳定、便于继续扩展的入口。
