# 游戏 Wiki 总目录

`Wiki/` 用来集中维护本项目的游戏资料页、系列索引和专题入口。这个首页只负责一级导航，不再罗列每一部作品的细项；想看某个系列当前已经整理到哪一作，直接进入对应目录里的 `README.md`。

## 目录说明

当前 `Wiki` 目录主要分为四类：

- 综合游戏系列：动作、射击、策略、竞速、RPG 等主机 / PC 系列
- `Gal` 分区：Galgame / 视觉小说相关系列与单作条目
- 中文 / 日式系列：中文游戏、JRPG、ADV 以及相关专题
- 专题入口：适合按时代或“未成稳定系列作品集”浏览的整理页

## 使用方式

推荐按下面方式浏览：

1. 想找知名系列，直接从“综合游戏系列目录”进入。
2. 想看视觉小说 / Galgame，直接进入 [《Gal》分区](./Gal/)。
3. 想看中文、JRPG 或 ADV 相关整理，使用“中文 / 日式系列目录”。
4. 想看时代概览或暂未形成稳定系列的作品集合，进入 [《辉煌时代》专题](./辉煌时代/README.md) 或 [《未有系列》专题](./未有系列/README.md)。

## 当前整理原则

- 总目录只保留一级入口，避免变成过长的作品总表
- 系列内作品覆盖情况以下级 `README.md` 为准
- `Gal` 分区单独维护，不再在总目录里拆散罗列
- 新增系列时优先补一级入口，减少总 README 频繁重写

一句话说，这份 README 更像馆内地图，负责把人带到正确的分区，而不是在首页展开所有词条。
