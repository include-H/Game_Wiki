# 《奥尔资讯》游戏 Wiki 目录

> 说明：本目录用于集中整理奥尔资讯多媒体股份有限公司在华语手机游戏时代推出的相关作品与子系列条目。本文主要依据公司公开资料、公开应用页面、巴哈姆特与相关怀旧资料整理而成。当前父目录先以《恋爱物语》这条最具代表性的 Java / 早期智能机恋爱 ADV 支线为起点，并已补入《猫恋物语－约定的彼方》《魔女恋歌》《星空恋曲～遥远的记忆～》这几部同厂近邻作品。

---

## 1. 系列概览

如果要整理华语 Java 手游时代的恋爱 / AVG 作品，奥尔资讯是一个很难绕开的名字。它的重要性并不在于拥有像主机大厂那样庞大的全球品牌，而在于它确实在功能机与早期智能机时期，持续推出过一批有明确叙事企图、题材风格也比较鲜明的中文手机恋爱游戏。

这些作品最值得单独立父目录整理的地方，在于它们之间既有明显共通气质，也未必都属于同一条严格编号主线。一部分作品可以直接放进《恋爱物语》系列内部理解，另一部分则更适合看成同厂、同时代、同题材方向下的近邻作品。也正因为如此，当前目录采用“父目录统筹 + 子系列 README + 单作 Wiki”的方式处理，而不是把所有标题硬塞进一条主线。

从今天回看，奥尔资讯这批作品很能代表一个已经过去的手机游戏阶段：单机叙事、短中篇流程、偶像剧式角色关系、以及在有限机能上努力做演出的中文恋爱 ADV。它们的历史价值，恰恰来自这种时代感。

## 4. 版本关系与补充说明

奥尔资讯父目录最容易混淆的地方，不是哪一作有没有重制，而是“哪些是严格同系列，哪些只是同厂近邻作品”。

当前需要先分清的几件事包括：

- 《恋爱物语》目前可以先作为一条相对独立的系列主干来整理
- 《猫恋物语》《魔女恋歌》《星空恋曲》等标题虽然在题材、年代和厂商上高度接近，但公开资料并未稳定显示它们属于《恋爱物语》编号正传
- 因此目录结构上更稳妥的做法，是先建立《奥尔资讯》父目录，再让《恋爱物语》作为一个子系列存在

---

## 5. 一句话总结

《奥尔资讯》父目录的意义，不是把所有同厂恋爱游戏混成一条主线，而是先把“同系列”和“同厂相关作品”这两个层级理顺。
