# 《恋爱物语》系列 Wiki 目录

> 说明：本目录用于整理台湾奥尔资讯在 Java 手机游戏时代推出的《恋爱物语》系列条目，当前聚焦《湛蓝的回忆》《我的机器人女友》《梦想之翼》这条最常被中文玩家并提的主干。本文主要依据奥尔资讯公开资料、公开应用页面、巴哈姆特 ACG 资料库、公开怀旧站点与相关玩家讨论整理而成。由于 Java 手游时代的官方页面留存有限，个别作品的首发日期与版本细节会在正文中注明“公开资料可确认到的范围”。本目录是 [《奥尔资讯》父目录](../README.md) 下的一个子系列。

## 1. 系列概览

如果要给华语 Java 手游时代找一批很能代表“青春感伤系恋爱 ADV”的作品，《恋爱物语》系列很难绕开。它不是靠复杂系统出名，也不是靠长线运营建立名声，而是靠几个当年手机平台上少见的卖点留下印象：完整的恋爱剧情、明确的女主角路线、偏偶像剧气质的桥段、以及在那个屏幕很小、机能有限的年代里仍努力做出的立绘表情、配音与多结局结构。

这一系列的核心气质并不在“模拟”二字本身，而在“把一段手机上的青春恋爱故事尽量演得像一部会让人记住的偶像剧”。从海边三角关系的《湛蓝的回忆》，到把未来恋人与日常经营揉在一起的《我的机器人女友》，再到把恋爱和明星梦想放在一起写的《梦想之翼》，它们共同构成了一条很典型的 2000 年代末至 2010 年代初中文手机恋爱游戏轨迹。

和后来偏抽卡、长线陪伴或社交驱动的恋爱手游不同，《恋爱物语》系列更接近“单机式手机文字冒险”。玩家买到、装上、通关，然后多年后只记得某个角色、某句台词、某个分歧结局给自己的情绪冲击。这也是它今天仍会被反复提起的原因。

## 2. 分期方式

当前目录按系列内部题材与时期，暂分为三个阶段：

### 2.1 起点作：海边青春伤感线

- [《恋爱物语：湛蓝的回忆》Wiki](./恋爱物语_湛蓝的回忆_WIKI.md)

这是系列最早被许多玩家记住的一作。它用海岛旅行、现任女友与神秘新相遇构成典型的青春三角关系，也奠定了系列“疗愈系手机偶像剧”的整体气质。

### 2.2 续作：未来恋人与经营强化线

- [《恋爱物语：我的机器人女友》Wiki](./恋爱物语_我的机器人女友_WIKI.md)

第二阶段把题材从海边青春恋曲转向近未来恋爱幻想。它一边保留系列的情感叙事，一边明显增加经营、服装与配音等更偏手机产品化的内容。

### 2.3 第三作：梦想与恋爱的舞台线

- [《恋爱物语3：梦想之翼》Wiki](./恋爱物语3_梦想之翼_WIKI.md)

第三阶段进一步把故事从单纯恋爱推进到“爱情与梦想”的并行结构。相较前两作，它更像是把恋爱手游叙事与演艺成长题材拼在一起的阶段性升级。

## 3. 阅读顺序

推荐按发售顺序阅读：

1. 《恋爱物语：湛蓝的回忆》
2. 《恋爱物语：我的机器人女友》
3. 《恋爱物语3：梦想之翼》

这种顺序最容易看出系列气质如何变化：先是纯粹的青春伤感恋爱剧，再转向带一点幻想色彩和经营要素的续作，最后发展到“恋爱 + 梦想追逐”的偶像剧式叙事。

## 5. 版本关系与补充说明

这个系列最容易混淆的，不是正传顺序，而是“Java 原版”和“后来的 Android 流通版”。

- 《湛蓝的回忆》公开资料可确认其 2009 年 8 月 1 日已以手机 AVG 恋爱游戏形态上线；后续又出现 Android 版本与商店页、评测页，因此今天网络上常见截图和介绍未必都来自最早的 Java 原版。
- 《我的机器人女友》通常被公开资料直接视作《恋爱物语》正宗续作；现存最容易找到的是后来的 Android 包与商店信息，但它的系列定位本身来自更早的 Java 时代。
- 《梦想之翼》是系列第三作，现存公开介绍多集中在 Android 转载或下载页；从描述看，它仍承接前两作的单机恋爱 ADV 传统，但题材明显转向“爱情与明星梦想”。
- 这三作没有明确留存出像主机游戏那样复杂的 DLC、资料片、导演剪辑版关系。真正需要说明的，是是否存在后续 Android 移植 / 重发，以及这些版本是否只是平台转移而非内容重构。
- 纯数值附赠、渠道包与安装包版本号，对理解系列本身的重要性远低于“原始题材定位”和“是否属于 Java 时代原作”。

## 7. 一句话总结

《恋爱物语》系列最值得记住的，不只是它曾让不少玩家在功能机和早期智能机上掉过眼泪，而是它把“单机恋爱文字冒险”这件事，认真地带进了华语 Java 手游时代。
