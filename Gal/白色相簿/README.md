# 《白色相簿》系列 Wiki 目录

> 说明：本目录用于集中整理《白色相簿》系列游戏 Wiki 条目。本文主要依据 AQUAPLUS 官方产品页、官方站点存档、Steam 商店页、Wikipedia 与相关公开资料整理而成，默认包含主线梗概、版本关系说明与一定剧透。

---

## 1. 系列概览

《白色相簿》是 Leaf / AQUAPLUS 旗下很有代表性的恋爱冒险游戏系列。它表面上常被归进“冬季恋爱视觉小说”这一类，但真正让它在中文圈长期留下名字的，并不是单纯的恋爱气氛，而是它特别擅长写一种会被现实磨损的关系: 名气、距离、时间差、误解、逞强和不肯坦白，都会把一段看似稳定的感情一点点推向裂缝。

系列目前最核心、也最值得优先整理的主干，其实只有两代。1998 年的初代《WHITE ALBUM》奠定了“艺能界与日常恋情交错”“冬天、歌曲与关系破裂感互相缠绕”的母题；2010 年开始展开的《WHITE ALBUM2》则把这一母题重新推到更强的戏剧结构里，写成了一部以三角关系、青春余波与成年后选择为核心的长篇。两代共享世界观与标题意象，但主角群并不相同，因此它更像“同一系列下的两次高峰”，而不是必须连续追完整个时间线才能理解的单一长篇。

如果只看产品形态，这个系列也很典型地体现了 Leaf / AQUAPLUS 老牌视觉小说的演变路径。初代先是 1998 年的 Windows 成人向作品，之后改编为家用机全年龄版，再在 2010 年被完整重制为《WHITE ALBUM -綴られる冬の想い出-》；《WHITE ALBUM2》则从 PC 章节式推出，后续整合为主机版《WHITE ALBUM2 幸せの向こう側》，再延伸出 `Mini-After Story` 与 `Extended Edition`。因此，理解这个系列时最重要的不是死记所有移植平台，而是先分清“初代原版 / 初代重制系”和“2代章节版 / 2代整合版”这两条线。

## 2. 分期方式

结合系列实际规模与版本结构，《白色相簿》可以先按下面两段理解：

### 2.1 初代与重制系

- [《WHITE ALBUM》Wiki](./WHITE_ALBUM_WIKI.md)

这一段对应 1998 年初代《WHITE ALBUM》以及后来的《WHITE ALBUM -綴られる冬の想い出-》。它是整个系列的原点，重点在于说明原版和重制版的关系，以及为何今天中文玩家讨论“白色相簿 1”时，多半实际接触的是重制系而不是 1998 原版。

### 2.2 《WHITE ALBUM2》章节整合系

- [《WHITE ALBUM2》Wiki](./WHITE_ALBUM_2_WIKI.md)

这一段对应 `Introductory Chapter`、`Closing Chapter`、主机整合版《幸せの向こう側》、`Mini-After Story` 与 `Extended Edition`。它不是单一一作就能讲清的作品，因此目录页会把它当成一个“以 2 代本体为核心、必须同时讲章节关系和整合关系”的条目处理。

## 3. 阅读顺序

如果是第一次接触《白色相簿》系列，比较推荐按下面的顺序理解：

1. 《WHITE ALBUM》
2. 《WHITE ALBUM2》

需要补充说明的是：

- 两代共享同一世界观脉络，但主角、核心冲突和故事完成度都是独立的，所以《WHITE ALBUM2》理论上可以单独入门。
- 不过如果先接触初代，再去读 2 代，会更容易理解这个系列为什么总绕着“歌”“冬天”“关系中越来越无法直说的部分”反复写。
- 初代今天最常见的入口并不是 1998 年原版，而是 2010 年起的重制版《WHITE ALBUM -綴られる冬の想い出-》及其后续 PC / Steam 发行版本。
- 《WHITE ALBUM2》内部则建议严格按 `Introductory Chapter -> Closing Chapter -> coda` 的逻辑理解；如果接触的是《WHITE ALBUM2 幸せの向こう側》或 `Extended Edition`，本质上也是在读这条顺序，只是载体从分章变成了整合版。

## 5. 版本关系与补充说明

- 1998 年的《WHITE ALBUM》是系列原点，由 Leaf 开发、AQUAPLUS 发行，最初是 Windows 成人向恋爱冒险游戏。
- 初代后续曾推出 PlayStation 全年龄版，但今天更关键的版本节点是 2010 年的重制作《WHITE ALBUM -綴られる冬の想い出-》。这不是简单移植，而是把初代完整改造成新演出规格的全年龄版。
- 《WHITE ALBUM -綴られる冬の想い出-》先在 2010 年登陆 PlayStation 3，2012 年推出 Windows 下载版，2023 年又以 `WHITE ALBUM: Memories like Falling Snow` 的名称在 Steam / DMM 上重新发行。
- 《WHITE ALBUM2》原始 PC 版分为 `Introductory Chapter` 与 `Closing Chapter` 两部分，通常讨论时还会把 `coda` 视为后半最关键的终局章节。
- 2012 年的《WHITE ALBUM2 幸せの向こう側》是很重要的整合节点。它把前两章整合进 PlayStation 3 平台，并加入新剧情、文本润色、额外 CG 与大量特典内容；2013 年又移植到 PS Vita。
- `WHITE ALBUM2 ミニアフターストーリー` 是 2014 年以应募赠品形式提供的后日谈性质补充内容；`WHITE ALBUM2 EXTENDED EDITION` 则于 2018 年在 Windows 平台把两章本篇、`Mini-After Story`、主机版追加剧情与相关附属内容一起整合，是理解 2 代版本关系时最完整的一版之一。
- 系列没有那种以“大型数值 DLC”或“服装包”构成的现代商业扩展层。真正重要的，是原版、重制版、整合版与后日谈补完之间的关系。

---

## 6. 一句话总结

《白色相簿》系列真正动人的地方，不在于它写了多少场恋爱，而在于它总在写: 当感情进入现实，冬天往往不是背景，而是那段关系本身的温度。
