# 《同级生 / 下级生》系列 Wiki 目录

> 说明：本目录用于集中整理 elf 以《同级生》《下级生》为核心的一组校园恋爱模拟名作条目。本文主要依据 FG REMAKE / Shiravune 等官方页面、Wikipedia、萌娘百科、MobyGames 与相关公开资料整理而成。由于这几部作品跨越 PC-98、DOS/V、Windows 与近年的重制版时代，正文会尽量把“原版首发”和“后续重制 / 移植”的关系拆开说明。本文包含系列结构说明与一定程度剧情概述。

---

## 1. 系列概览

如果把 1990 年代日本美少女游戏的发展史压缩成几条最关键的脉络，《同级生》与《下级生》几乎一定会被放进同一段里谈。它们并不完全是同一个连续剧情 IP，却共享了非常接近的设计谱系：自由移动的日程推进、围绕校园与城市生活展开的多女主攻略、以及“恋爱模拟不只是选项分支，而是要在一段可流动的时间里做行动管理”的核心思路。

《同级生》在 1992 年把这种结构第一次做成了明确的代表作，《下级生》则在 1996 年把它进一步扩写成规模更大的群像恋爱模拟；《同级生2》在 1995 年把自由探索与人物关系推进到更成熟的阶段，《下级生2》则在 2004 年以更晚近的视觉小说语法重做了一次“长篇校园群像恋爱”。

所以，这个目录采用“同级生 / 下级生合并整理”的方式，并不是把它们强行当成单一世界观，而是把它们看作 elf 校园恋爱模拟系谱里最值得并读的主干。读者真正需要分清的，是每一作各自在这条系谱里的独特定位，以及原版、移植版、重制版之间到底是什么关系。

当前目录先按“奠基期 -> 成熟期 -> 重制回归期”来理解：

- [《同级生》Wiki](./同级生_WIKI.md)
- [《下级生》Wiki](./下级生_WIKI.md)

这一阶段最重要的意义，是把“校园恋爱模拟 + 自由行动管理”从概念做成一套真正能成立的产品形式。《同级生》是起点，《下级生》则是第一次大规模扩写。

### 2.2 成熟期

- [《同级生2》Wiki](./同级生2_WIKI.md)

- [ ] 《同级生 Remake》

## 3. 阅读顺序

如果是第一次接触这条系谱，比较推荐按下面顺序阅读：

1. 《同级生》
2. 《下级生》
3. 《同级生2》
4. 《下级生2》

这样读的好处，是能先看到“这一套玩法是怎么被发明出来的”，再看它如何被扩充和细化。

- 《同级生》最适合当作起点，因为它决定了这类作品后来最常被提起的基本语法。
- 《下级生》虽然不是《同级生》的剧情续集，但它能让人立刻看见 elf 如何把同类设计做得更大、更复杂。
- 《同级生2》是这条系谱里最常被视为巅峰讨论对象的一作，放在前两作之后看，能更清楚感受到它为什么会被反复提起。
- 《下级生2》年代更晚，文本节奏、画风和整体气质都已不再是纯 PC-98 时代的样子，适合放在最后看作“这一系作品在 2000 年代中的再展开”。

## 5. 版本关系与补充说明

- 《同级生》原版首发于 1992 年 PC-9801，后有 DOS/V、FM TOWNS、X68000、Windows 版与家用机移植；近年又有 `同級生リメイク` 与主机版 `CSver`。今天讨论本作时，必须先分清“1992 年原作”和“2021 年后重制系”不是同一个层级。
- 《下级生》原版首发于 1996 年 PC-9801，后移植到 Windows、世嘉 Saturn、PlayStation 等平台；FG REMAKE 又在 2025 年推出《下級生リメイク》，并附带可在现代 Windows 上运行的 DOS/V 版《下級生》。这让它的“原版体验”和“现代重制体验”需要分开描述。
- 《同级生2》原版一般以 1995 年 PC-9801 版作为起点，后续 Windows 95 版在今天的讨论里反而更常被拿来当基准；2024 年发售的《同級生2リメイク》官方明确写明是以 Windows 95 版为基础进行现代化重制。
- 《下级生2》首发于 2004 年 Windows 平台，年代已经明显晚于前三作，因此它和《同级生》《下级生》那种 PC-98 / DOS/V 气质并不完全相同。它目前最需要说明的版本关系，不是 DLC，而是后来的全年龄移植、家用机版与重制企划传闻并未真正取代原版主体验。
- 这个系谱里真正值得写清的“追加内容”，通常不是服装或数值 DLC，而是重制版、全年龄版、家用机移植版与打包复刻版如何影响玩家今天接触这些作品的方式。

## 7. 一句话总结

《同级生》与《下级生》最值得并读的地方，不是它们共享同一条剧情，而是它们一起把日本校园恋爱模拟最经典的一套时间管理语法，完整地立了起来。
