# 《fumi》作品 Wiki 目录

> 说明：本目录用于集中整理日本独立开发者 fumi（App Store 页面开发者名显示为 `mafumi yoshida`）的叙事向游戏条目。本文主要依据 App Store、Steam、room6 官方新闻稿、KADOKAWA 漫画资料与相关公开 Wiki 整理而成，默认包含作品定位、版本关系与一定剧透。

---

## 1. 开发者概览

如果把《我在7年后等着你》放回独立游戏语境去看，fumi 这条线最特别的地方，并不是做了多复杂的系统，而是用一种很节制的形式，把“手机端也可以承载完整叙事冒险”这件事做得非常清楚。它最早是一部 2017 年上线的移动端作品，靠免费本篇、短章节推进和强剧情钩子迅速传播；到了 2021 年，又通过 room6 的发行进入 Nintendo Switch 与 Steam，把原本偏手机阅读感的作品，进一步固定成一部可被主机 / PC 玩家重新认识的独立叙事游戏。

从今天回头看，fumi 这条线的代表性，主要就集中在《我在7年后等着你》这部作品身上。它既有明显的个人创作气质，也具备那种少见的“低成本外观下却有完整情绪回收和结构野心”的特征。对这个目录来说，先把这部代表作写扎实，比急着铺开更多周边条目更重要。

## 3. 版本关系与补充说明

当前理解这部作品时，最重要的是先分清两个层级：

- 2017 年的 iOS / Android 版，是作品原始传播形态。本篇可以直接通关，后续补完内容则通过更新与附加付费项逐步加入。
- 2021 年的 Nintendo Switch 与 Steam 版，不是剧情重制，而是把移动端后期补入的 `Epilogue`、`Side Story`、人物 / 术语辞典与 `Sound Mode` 以买断制形式整合进来。

对中文玩家来说，还需要再补一层语境：

- 原版 App Store 页面明确标注过简体中文翻译由扑家吧提供。
- 截至 2026 年 3 月，可见到中国大陆 App Store 上由 Pujia8 Limited 发布的单独版本《我在7年后等着你 - 7年的约定》。

也就是说，这部作品的“中文传播史”和“平台版本史”是交错在一起的，整理时不能简单把它视作只有一个固定版本名的单一产品。
