# 使命召唤系列 Wiki 目录

> 说明：本目录用于集中整理《使命召唤》系列相关 Wiki 条目。当前已覆盖从二战时期原点、现代战争子系列、黑色行动子系列到 Ghosts、Advanced Warfare、Infinite Warfare 等主要分支内容。后续可继续补充尚未整理的作品、重制版、手游与 Warzone 相关条目。

---

## 1. 系列概览

《使命召唤》（Call of Duty）是 21 世纪最具影响力的第一人称射击游戏系列之一。它最初从二战题材起家，后来逐步扩展出现代战争、冷战谍战、近未来科技战争与多人在线服务型内容等多条路线，并在很长时间里保持着年度级主流影响力。

如果按当前目录里的条目来看，这个系列大致可以拆成几条核心脉络：

- 二战原点与经典战争阶段
- 《现代战争》子系列
- 《黑色行动》子系列
- 独立分支作品，如《幽灵》《高级战争》《无限战争》
- 近年重启与新世代阶段

## 3. 推荐阅读顺序

如果是想按系列发展脉络来阅读，比较适合按下面顺序：

1. 二战原点作品
2. 《现代战争》子系列
3. 《黑色行动》子系列
4. 《幽灵》《高级战争》《无限战争》等独立路线
5. 再回头看近年重启阶段作品

如果是想按系列内部世界线或气质来读，可以简单理解为：

- 想看系列原点和战争题材根基，就先看二战阶段。
- 想看现代军事大片与全球危机叙事，就看《现代战争》。
- 想看冷战、阴谋、记忆操控与 Treyarch 风格，就看《黑色行动》。
- 想看更偏概念化、实验性或未来感的路线，就看《幽灵》《高级战争》《无限战争》。
