## 1. 系列概览

《雷电》是街机时代最有代表性的纵版卷轴射击系列之一，也是从 Seibu Kaihatsu 一路延续到 MOSS 时代的老牌 STG 名字。它最容易被记住的地方，是战机造型、火力升级、红蓝紫武器切换、密集却不至于完全失控的弹幕，以及那种非常典型的“人类战机迎击外星晶体军”的硬派街机气质。

和很多完全靠关卡体验驱动的老牌射击游戏相比，《雷电》虽然剧情表达一直都不算重，却不是完全没有连续主线。系列长期围绕神秘外星生命“Crystals”、地球联合反击力量 `VCD` 与主力战机 `Fighting Thunder / Raiden` 展开，只是叙事通常更多通过开场设定、Boss 核心、续作危机升级和现代版本说明来呈现。

也正因为它既有足够长的主线作品链，又夹着 `Raiden Fighters`、现代强化版和《Raiden NOVA》这类分支，所以当前目录采用“系列总览 + 合并条目”的写法。

## 4. 一句话总结

《雷电》最值得被记住的，是它把“看上去很朴素的纵版打飞机”做成了一条横跨街机、主机和现代移植时代都还成立的老牌硬核射击系列。
