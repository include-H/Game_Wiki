## 1. 系列概览

《Modern Combat》通常可直译为“现代战争”，是 Gameloft 在智能手机与功能机向触屏时代过渡期打造的一条现代军事第一人称射击系列。它的特殊位置，在于它并不是后来主机大作的简单附属移植，而是 2009 年前后少数真正尝试把“主机式现代军事 FPS 演出、战役节奏与多人对战”完整搬到移动端的代表性品牌。

如果从系列演化来看，这条线大致经历了三个阶段：

- 起步阶段：以《Modern Combat: Sandstorm》确定系列基调，证明触屏设备也能承载相对完整的现代军事 FPS。
- 成长阶段：以《Modern Combat 2》《Modern Combat 3》把战役规模、在线对战和“移动端大片感”一起做大。
- 转型阶段：从《Modern Combat 4》到《Modern Combat 5》，再到偏英雄射击的《Modern Combat Versus》，系列逐步从早期单机战役导向走向更强的在线服务与职业化设计。

它在系列史上的价值，不只是“Gameloft 版现代战争”，而是早期移动端射击游戏如何主动追赶主机时代工业表达的一条很清晰的样本。

## 3. 阅读顺序

如果想按系列自然发展去读，推荐顺序如下：

1. 《Modern Combat: Sandstorm》
2. 《Modern Combat 2: Black Pegasus》
3. 《Modern Combat 3: Fallen Nation》
4. 《Modern Combat 4: Zero Hour》
5. 《Modern Combat 5: Blackout》
6. 《Modern Combat Versus》

如果只想先理解这个系列为什么在当年有代表性，那么前五作已经基本能把它的主脉络看清：

- 初代负责把“移动端现代军事 FPS”这个框架立起来。
- 二代负责把多人对战和全球战场格局做得更像完整续作。
- 三代则把整体气质推向更强的灾难大片与高强度在线成长。
- 四代把系列前期的画面、演出和多人结构推到一个完成度高峰。
- 五代则把整条路线正式扳向长期在线化和服务型 FPS。

## 5. 版本关系与补充说明

需要先说明几件容易混淆的事：

- 《Modern Combat》系列的早期作品以 iPhone / iPad / Android 等移动平台版本为主，不同年份常会出现 `HD`、平板版、不同商店版或面向不同系统的移植版。
- 这些 `HD` 或平板强化版本，通常属于同一部作品在不同设备上的适配或增强展示，不应简单当作全新独立正传。
- 早期几作虽然都带有明显的现代军事题材与连续的品牌气质，但剧情连续性并不像某些主机三部曲那样完全以固定人物长线推进；更重要的是看它们在系统、规模和移动端技术表现上的递进。
- 《Modern Combat 5: Blackout》之后，系列方向明显更偏在线化；到了《Modern Combat Versus》，核心结构已经不再是传统的单人战役 + 经典多人 FPS 组合，而是更偏英雄射击的多人对抗。

就目前目录的整理原则而言，后续会优先把“对理解系列主体演化最关键”的作品补齐，再视情况决定是否单独拆分特殊商店版、高清版或停运后的历史说明页。

《Gameloft 现代战争》系列最值得记住的，不只是它像不像某些主机 FPS，而是它曾经非常直接地定义了“早期智能手机时代，现代军事射击游戏究竟能被做成什么样”。
