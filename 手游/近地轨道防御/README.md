# 《近地轨道防御》系列 Wiki 目录

> 说明：本目录用于集中整理 Gameloft 科幻第一人称射击系列《N.O.V.A. / Near Orbit Vanguard Alliance》相关 Wiki 条目。当前先完成系列前三部正传，即《N.O.V.A. - Near Orbit Vanguard Alliance》《N.O.V.A. 2: The Hero Rises Again》与《N.O.V.A. 3》，后续再补《N.O.V.A. Legacy》以及系列平台演化与移动端科幻 FPS 路线专题。

---

## 1. 系列概览

《N.O.V.A.》通常被中文玩家称作《近地轨道防御》，全名是 `Near Orbit Vanguard Alliance`。它是 Gameloft 在智能手机时代早期打造的一条科幻第一人称射击系列，也是与《Modern Combat》并列、但气质明显不同的另一条旗舰 FPS 路线。

如果说《Modern Combat》更偏现代军事大片，那么《N.O.V.A.》代表的就是 Gameloft 对“掌上设备版科幻太空战争”的理解。轨道殖民地、外星种族、动力装甲、人工智能搭档、舰船与废墟战场，共同构成了这条系列的核心识别度。它长期被拿来与《Halo》比较，但它真正重要的地方，并不只是相似，而是它确实在 2009 年前后把一套完整的科幻 FPS 大作感搬到了移动端。

从系列演化来看，这条线大致可以分成三个阶段：

- 起点阶段：初代《N.O.V.A.》负责把“移动端科幻 FPS”这一框架立起来。
- 成长阶段：《N.O.V.A. 2》把战役规模、多人联机与系列叙事继续向上扩张。
- 成熟阶段：《N.O.V.A. 3》则把画面、场面和大灾变气质推向系列前期高峰。

后来的《N.O.V.A. Legacy》则更像围绕初代重新包装、面向新一代手机环境推出的轻量化重做作品，适合单独说明，不应直接和前三作正传混写。

## 3. 阅读顺序

如果想按系列自然发展来读，推荐顺序如下：

1. 《N.O.V.A. - Near Orbit Vanguard Alliance》
2. 《N.O.V.A. 2: The Hero Rises Again》
3. 《N.O.V.A. 3》
4. 《N.O.V.A. Legacy》

如果只想先理解这个系列为什么在当年有代表性，那么前三作就已经足够看出主体轮廓：

- 初代负责把掌上设备上的科幻 FPS 框架立起来。
- 二代把战役规模、多人模式和 Kal Wardin 这一主角线继续做大。
- 三代则把系列推向更强的末世战场感与更夸张的移动端视听规格。

## 5. 版本关系与补充说明

在写这个系列时，有几件事需要先说明：

- 早期《N.O.V.A.》作品大多围绕 iPhone / iPad / Android 等移动平台扩展，也出现过 webOS、bada、PSN Minis、Mac OS X、BlackBerry、Windows Phone 等不同设备版本。
- 不同平台上的 `HD`、平板版、移植版，通常仍属于同一部作品在不同硬件环境下的发行，不应轻易拆成独立正传。
- 《N.O.V.A. Legacy》虽然沿用了 Kal Wardin、Yelena 与初代品牌名义，但更适合视作重做 / 轻量重启性质作品，而不是严格顺排在《N.O.V.A. 3》后的传统续作。
- 这一系列长期带有很明显的《Halo》式审美影响，但写条目时仍应以其自身在移动端的发展路径、系统升级和平台语境为主，不把“相似”直接写成全部结论。

《近地轨道防御》系列最值得记住的，不只是它像不像《Halo》，而是它曾经非常直接地定义了“智能手机时代早期，科幻第一人称射击游戏究竟能被做成多接近大作”。
