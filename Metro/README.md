# 《地铁》系列 Wiki 目录

> 说明：本目录用于集中整理《地铁》（Metro）系列相关 Wiki 条目。当前已覆盖《地铁：2033》《地铁：最后的曙光》与《地铁：离去》三部主线作品，后续可继续补重制版本差异、资料片与小说和游戏世界观关系等内容。

---

## 1. 系列概览

《地铁》（Metro）是由乌克兰工作室 4A Games 开发的末世题材第一人称射击 / 生存叙事系列，其世界观改编自 Dmitry Glukhovsky 的同名小说体系。系列以核战后的莫斯科地铁网络为核心舞台，把生存恐惧、政治派系、宗教意味、资源匮乏和超自然异象混合在一起，形成了一种非常鲜明的“废墟科幻 + 地下寓言”气质。

如果从当前目录已整理和后续可扩展的角度看，《地铁》系列大致可以分成几段：

- 《地铁：2033》奠定世界观与氛围基底的起点阶段
- 《地铁：最后的曙光》承接前作结局并扩展派系与人物关系的中段
- 《地铁：离去》把系列从地下封闭空间推进到更大尺度荒原旅程的转型阶段
- `Redux` 版本作为技术与体验层面的统一重制整理

## 3. 推荐阅读顺序

如果是想按作品推出与世界观展开顺序理解这个系列，比较自然的阅读顺序是：

1. 《地铁：2033》
2. 《地铁：最后的曙光》
3. 《地铁：离去》

需要说明的是：

- `Metro 2033 Redux` 与 `Metro: Last Light Redux` 更适合视为体验层面的重制整理版本，而不是完全独立的新剧情作品。
- 游戏系列虽然改编自 Dmitry Glukhovsky 的小说世界，但并不是逐字逐句照搬；它更像是在保留核心框架后，做出的互动化重构。
