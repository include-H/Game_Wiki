# 蝙蝠侠游戏 Wiki 目录

> 说明：本目录用于集中整理当前项目中的蝙蝠侠相关游戏 Wiki 条目。当前内容以《阿甘》系列为主体，也开始收录《蝙蝠侠：故事版》这类独立路线作品，后续可继续扩展到掌机、VR、衍生作与专题条目。

---

## 1. 整体概览

蝙蝠侠游戏在近二十年里大致可以分成几条比较清晰的路线。其中最有影响力的当然是《蝙蝠侠：阿甘》系列，它以高完成度的近战、潜行、侦探和哥谭氛围建立起现代超级英雄动作游戏的标杆；但除此之外，也存在像 Telltale 这样更偏互动叙事、人物关系和道德抉择的独立版本。

如果按当前目录里的内容来大致区分，可以先理解成两条主要脉络：

- 《蝙蝠侠：阿甘》系列
- Telltale 互动叙事路线

## 3. 推荐阅读顺序

如果是想读《阿甘》主线，比较适合按下面顺序：

1. 《阿甘起源》
2. 《阿卡姆疯人院》
3. 《阿甘之城》
4. 《阿甘骑士》

如果是更想按实际影响力和玩法演化来读，也可以先从《阿卡姆疯人院》开始，再顺着《阿甘之城》《阿甘骑士》一路读下去，最后回头补《阿甘起源》这部前传。

如果是想看阿甘之外的独立诠释路线，则可以按下面顺序阅读：

1. 《蝙蝠侠：故事版》
2. 《蝙蝠侠：内敌》
