---

## 1. 系列概览

《孤岛危机》（Crysis）是 2000 年代后期到 2010 年代前期非常有代表性的科幻第一人称射击系列。它最早因初代作品夸张的图形表现和半开放式热带战场而出名，随后又通过《孤岛危机 2》把舞台从孤岛推进到崩坏纽约，再由《孤岛危机 3》完成对 Prophet、CELL 与 Ceph 主线的收束。

如果按当前目录已经整理的内容来看，这个系列大致可以分成几段：

- 初代《Crysis》作为“显卡危机”时代象征的技术型起点
- 《Crysis Warhead》作为并行视角的强化型独立资料片
- 《Crysis 2》作为系列走向主流多平台、转入都市末日战场的关键转折
- 《Crysis 3》作为回收前两代气质并完成三部曲收束的终章

## 3. 推荐阅读顺序

如果是想按系列剧情和气质变化来阅读，比较适合按下面顺序：

1. 《孤岛危机》初代
2. 《孤岛危机：弹头》
3. 《孤岛危机 2》
4. 《孤岛危机 3》

如果是想按“系列风格怎么变化”来读，也可以这样理解：

- 想看系列最早的技术冲击、热带战场和纳米服起点，就从初代《Crysis》开始。
- 想看初代核心手感如何被压缩得更紧、更猛，就接着读《弹头》。
- 想看这个系列怎样从 PC 技术传奇转向主流都市科幻大片，就重点读《孤岛危机 2》。
- 想看三部曲如何把自由猎杀感、终章叙事和 Prophet 命运线收回来，就读《孤岛危机 3》。
