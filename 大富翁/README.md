## 1. 系列概览

《大富翁》是华语电子游戏史里最具代表性的“桌游式派对游戏”系列之一。它最容易被理解的地方，是玩家掷骰前进、买地盖屋、收租、用卡片互相算计；但它真正特别的地方，不只是把 `Monopoly` 电子化，而是把这一整套房产经营和破产竞争的桌游逻辑，改造成更适合电脑与主机环境的本土化合家欢游戏。

从 1989 年的初代开始，这个系列就把“地块买卖”做成了极容易理解的核心循环。此后各代又不断往里面加入股票市场、道具卡、神仙附身、地图事件、角色技能、多人热座与轻量剧情包装，慢慢形成了一种非常鲜明的《大富翁》式语言。对很多中文玩家来说，它既是桌游规则的电子化入门，也是朋友、家人或同学围着一台电脑轮流对局的共同记忆。

如果把《大富翁》看成一条演化链，真正重要的不是每一代剧情是否相连，而是这个系列怎样把“买地收租”的桌游祖型，一步步变成更热闹、更随机、更有本地文化气味的电子游戏派对玩法。

## 5. 版本关系与补充说明

- 《大富翁》系列主线通常指从 1989 年起的大宇 / 软星编号作品；但玩家记忆最深的一段，通常集中在《大富翁 3》《大富翁 4》《大富翁 6》《大富翁 7》《大富翁 8》到后来重新回归单机路线的《大富翁 10》《大富翁 11》。
- 这个系列并不是简单照搬 `Monopoly`。它保留了掷骰前进、地产交易、收租与破产淘汰这些祖型结构，但很早就加入了股票、卡片、神仙、新闻事件、医院 / 监狱、角色互动等电子游戏化改造。
- 《大富翁 4》及其后续强化版本《大富翁 4 Fun》常被视作系列最具公共记忆的一代；后者不是编号正传续作，而更接近在原有框架上扩充人物、地图与模式的强化版。
- 《大富翁 9》明显带有移动化、联网化和轻量付费设计取向，在系列内部属于路线偏移较大的一作；而《大富翁 10》《大富翁 11》更像是重新强调单机 / 本地多人聚会属性的回归。
- 《大富翁》系列虽然也常给角色加上轻度冒险背景或“幸运之星”式包装，但它的重点从来不在连续大剧情，而在地图规则、随机事件和多人对局过程中自然长出来的戏剧性。

## 7. 一句话总结

《大富翁》系列最特别的地方，是它没有停留在“把 Monopoly 搬进电脑”，而是把这套买地收租的规则，做成了一种带着华语本土气味、非常适合多人互坑取乐的电子游戏玩法。
