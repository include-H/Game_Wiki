## 1. 系列概览

《黑手党》（Mafia）是一个以 20 世纪不同阶段美国黑帮史和有组织犯罪叙事为核心的动作冒险系列。和很多同类开放城市犯罪游戏相比，它最鲜明的特点并不是“自由度更大”，而是始终更强调线性叙事、时代氛围、人物命运和犯罪片气质。

如果按当前已经整理和准备整理的内容来看，这个系列大致可以分成两条主脉络：

- 原三部曲：`Mafia`、`Mafia II`、`Mafia III`
- 现代重制 / 整合阶段：`Mafia: Definitive Edition`、各作 `Definitive Edition`、`Mafia: Trilogy`

而在 2025 年，系列还进一步推出了前传性质的新作《Mafia: The Old Country》，把时间线拉回 20 世纪初的西西里，开始补写“黑手党”这一组织文化最早的源头想象。

从整体气质上说，《黑手党》系列最值得记住的地方，是它更像一组可以互动的黑帮电影。它当然有驾驶、枪战和城市探索，但真正让它成立的，一直是时代感极强的城市、逐步走向毁灭的主角人生，以及那种“加入黑帮不等于成为赢家，而更像被卷入一条很难回头的命运线”的悲剧色彩。

## 3. 推荐阅读顺序

如果想按发售顺序理解系列发展，比较适合这样读：

1. 《黑手党》（2002）或《黑手党：最终版》
2. 《黑手党 II》
3. 《黑手党 III》
4. 《黑手党：故乡》

如果想按系列内部时间线大致理解，也可以这样看：

1. 《黑手党：故乡》
2. 《黑手党》（2002）或《黑手党：最终版》
3. 《黑手党 II》
4. 《黑手党 III》

不过需要注意，《故乡》虽然是前传，但它是 2025 年才推出的新作，气质和制作条件都更接近现代作品，因此阅读或游玩时未必一定要按时间线先看它。对大多数人来说，仍然更适合先从 Tommy、Vito、Lincoln 这条经典三部曲脉络开始。

## 4. 版本关系说明

《黑手党》系列最容易混淆的地方，主要集中在 2020 年前后的重制与整合版本：

- 初代《Mafia》（2002）在 2020 年迎来从零重做的《Mafia: Definitive Edition》
- 《Mafia II: Definitive Edition》是基于原作的 HD remaster，而不是像初代那样彻底重做
- 《Mafia III: Definitive Edition》则主要是本体与 DLC 的整合再发行
- `Mafia: Trilogy` 把 `Mafia: Definitive Edition`、`Mafia II: Definitive Edition`、`Mafia III: Definitive Edition` 统一打包成一套完整合集

也就是说，讨论“黑手党 1 重制”时，最好明确区分：

- 2002 年原版《Mafia》
- 2020 年完全重做的《Mafia: Definitive Edition》

而讨论二代、三代时，则更适合理解为：

- 原版本体
- 后来的最终版 / 整合版 / 高清版

## 5. 当前覆盖范围说明

当前目录优先覆盖的是《黑手党》系列最核心的主干：

- Tommy Angelo 的禁酒令时期故事
- Vito Scaletta 的战后黑帮上升线
- Lincoln Clay 的 1968 复仇与重建线
- 以及《故乡》对系列起源神话的回溯

这样处理的好处，是能先把系列最重要的人物与时代感立起来，再决定是否补更多 DLC 细节、合集版本差异或更细的时间线专题。
