# 未有系列 Wiki 目录

> 说明：本目录用于整理那些"没有系列归属"的独立单作游戏。这些游戏可能是独立开发的作品，也可能是商业失败未能延续的作品，或者是开发者只打算做一部的作品。它们没有系列可以归入，但同样值得被记录。

---

## 1. 目录概览

"未有系列"收录的是那些独立存在的游戏作品。它们不像《使命召唤》《最终幻想》那样有系列传承，也不像《仙剑奇侠传》《轩辕剑》那样有明确的系列归属。

这些游戏可能因为以下原因被归入此处：

- 独立开发，本就是单作
- 商业表现不佳，未能延续
- 开发者解散，系列中断
- 题材特殊，难以系列化
- 开发者主动选择只做一部

但这并不意味着它们不重要。相反，许多独立单作在游戏史上有着独特的地位，它们可能是一次大胆的尝试、一个独特的创意、或者一段值得铭记的历史。

## 2. 当前已覆盖作品

### 2.1 《抗日：血战上海滩》（2003）

- [《抗日：血战上海滩》Wiki](./抗日血战上海滩_WIKI.md)

一款以抗日战争为题材的国产 FPS 游戏，2003 年由欢乐亿派开发发行。它是中国游戏史上少有的抗战题材 FPS，虽然技术上无法与同时期的国外大作相比，但它代表了中国游戏开发者的一次勇敢尝试。

**核心主题**：
- 抗日战争题材
- 国产 FPS 的早期尝试
- 淞沪会战历史背景
- 民族情感与游戏体验

### 2.2 《暴力摩托》系列（1991-1999）

- [《暴力摩托》Wiki](./暴力摩托_WIKI.md)

EA 开发的摩托车竞速 + 格斗游戏系列，1991 年至 1999 年间共发布 8 部正作。游戏将竞速与格斗结合，玩家在驾驶摩托车飞驰的同时，可以用拳打、脚踢、链条等方式攻击对手。这种"暴力竞速"的设计在当时非常前卫，成为了许多中国玩家的童年记忆。

**核心主题**：
- 竞速与格斗的独特结合
- 公路狂飙的刺激感
- 多人分屏对战的乐趣
- 摇滚乐配乐

### 2.3 《节奏天国 Wii》（2011）

- [《节奏天国 Wii》Wiki](./节奏天国Wii_WIKI.md)

任天堂于 2011 年在 Wii 平台发布的音乐节奏游戏，由 Nintendo SPD 开发、淳君监制。游戏采用极简的一键操作，配合各种创意十足的节奏小游戏（给猴子打针、生产威化饼干、打羽毛球……），创造出荒诞幽默又令人上瘾的游戏体验。是"简约节奏游戏"的代表作。

**核心主题**：
- 极简的一键操作
- 创意节奏小游戏
- 荒诞幽默风格
- 淳君监制的原创音乐
- 像素艺术风格

## 3. 后续可收录方向

如果后续继续扩展这个目录，可以考虑收录以下类型的作品：

**国产独立单作**：
- 《光明与黑暗》系列之外的单作
- 各种尝试性作品
- 小众但有意义的作品

**国外独立单作**：
- 独立游戏佳作
- 实验性作品
- 作者型游戏

**特殊题材作品**：
- 教育/严肃游戏
- 艺术/实验游戏
- 小众题材游戏

---

## 4. 收录标准

归入"未有系列"的作品，通常满足以下条件之一：

1. 没有系列归属，是独立单作
2. 原本计划系列化，但因各种原因中断
3. 开发者只打算做一部作品
4. 题材或形式特殊，难以系列化
5. 在游戏史上有独特意义，值得单独记录

---

## 5. 一句话总结

"未有系列"收录的是那些独立存在的游戏作品，它们没有系列传承，但同样值得被铭记。
