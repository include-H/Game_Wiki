# 未有系列 Wiki 目录

> 说明：本目录用于整理那些"没有系列归属"的独立单作游戏。这些游戏可能是独立开发的作品，也可能是商业失败未能延续的作品，或者是开发者只打算做一部的作品。它们没有系列可以归入，但同样值得被记录。

---

## 1. 目录概览

"未有系列"收录的是那些独立存在的游戏作品。它们不像《使命召唤》《最终幻想》那样有系列传承，也不像《仙剑奇侠传》《轩辕剑》那样有明确的系列归属。

这些游戏可能因为以下原因被归入此处：

- 独立开发，本就是单作
- 商业表现不佳，未能延续
- 开发者解散，系列中断
- 题材特殊，难以系列化
- 开发者主动选择只做一部

但这并不意味着它们不重要。相反，许多独立单作在游戏史上有着独特的地位，它们可能是一次大胆的尝试、一个独特的创意、或者一段值得铭记的历史。

## 2. 当前已覆盖作品

### 2.1 《抗日：血战上海滩》（2003）

- [《抗日：血战上海滩》Wiki](./抗日血战上海滩_WIKI.md)

一款以抗日战争为题材的国产 FPS 游戏，2003 年由欢乐亿派开发发行。它是中国游戏史上少有的抗战题材 FPS，虽然技术上无法与同时期的国外大作相比，但它代表了中国游戏开发者的一次勇敢尝试。

**核心主题**：
- 抗日战争题材
- 国产 FPS 的早期尝试
- 淞沪会战历史背景
- 民族情感与游戏体验

### 2.2 《暴力摩托》系列（1991-1999）

- [《暴力摩托》Wiki](./暴力摩托_WIKI.md)

EA 开发的摩托车竞速 + 格斗游戏系列，1991 年至 1999 年间共发布 8 部正作。游戏将竞速与格斗结合，玩家在驾驶摩托车飞驰的同时，可以用拳打、脚踢、链条等方式攻击对手。这种"暴力竞速"的设计在当时非常前卫，成为了许多中国玩家的童年记忆。

**核心主题**：
- 竞速与格斗的独特结合
- 公路狂飙的刺激感
- 多人分屏对战的乐趣
- 摇滚乐配乐

### 2.3 《超凡：双生》（2013）

- [《超凡：双生》Wiki](./超凡双生_WIKI.md)

Quantic Dream 于 2013 年推出的互动电影式叙事游戏，由 Elliot Page 与 Willem Dafoe 参与演出。作品围绕 Jodie Holmes 与超自然实体 Aiden 的共生关系展开，以非线性章节拼接她从童年到成年的生命轨迹，是 2010 年代 3A 互动叙事游戏的重要代表作之一。

**核心主题**：
- 互动电影与分支叙事
- 超自然能力与成长创伤
- 非线性人生结构
- 表演捕捉与电影化演出

### 2.4 《底特律：化身为人》（2018）

- [《底特律：化身为人》Wiki](./底特律化身为人_WIKI.md)

Quantic Dream 于 2018 年推出的互动电影式叙事游戏，以 2038 年的底特律为舞台，通过 Connor、Kara 与 Markus 三条主线并行讲述安卓人觉醒、社会恐慌与权利冲突。它凭借高密度分支、章节流程图与角色可死亡推进机制，成为互动叙事游戏中最具传播度的代表作之一。

**核心主题**：
- 人工智能与主体性
- 多主角分支叙事
- 近未来社会危机
- 选择后果与流程图结构

### 2.5 《暴雨》（2010）

- [《暴雨》Wiki](./暴雨_WIKI.md)

Quantic Dream 于 2010 年推出的互动电影式心理惊悚游戏，围绕“折纸杀手”案件展开，通过 Ethan Mars、Scott Shelby、Norman Jayden 与 Madison Paige 四名角色交叉推进。它以高压悬疑、失败也能继续的叙事设计，以及强烈的成人题材表达，成为互动叙事游戏迈入主流讨论的重要里程碑。

**核心主题**：
- 心理惊悚与连环杀手悬疑
- 多主角交叉叙事
- 失去、愧疚与父爱试炼
- 失败推进与分支结局

### 2.6 《节奏天国 Wii》（2011）

- [《节奏天国 Wii》Wiki](./节奏天国Wii_WIKI.md)

任天堂于 2011 年在 Wii 平台发布的音乐节奏游戏，由 Nintendo SPD 开发、淳君监制。游戏采用极简的一键操作，配合各种创意十足的节奏小游戏（给猴子打针、生产威化饼干、打羽毛球……），创造出荒诞幽默又令人上瘾的游戏体验。是"简约节奏游戏"的代表作。

**核心主题**：
- 极简的一键操作
- 创意节奏小游戏
- 荒诞幽默风格
- 淳君监制的原创音乐
- 像素艺术风格

### 2.7 《恶霸鲁尼》（2006）

- [《恶霸鲁尼》Wiki](./恶霸鲁尼_WIKI.md)

Rockstar Vancouver 于 2006 年推出的校园开放世界动作冒险游戏，以 Bullworth Academy 为舞台，让 Jimmy Hopkins 在一整套荒诞、残酷又幼稚的校园阶级秩序中一路闯出自己的位置。它既是 Rockstar 最特殊的 cult classic 之一，也是“开放世界不一定非得讲黑帮和枪战”的鲜明案例。

**核心主题**：
- 校园开放世界
- 青春期阶层冲突
- 讽刺与荒诞校园秩序
- 单作 cult classic 地位

### 2.8 《绯红结系》（2021）

- [《绯红结系》Wiki](./绯红结系_WIKI.md)

Bandai Namco Studios 与 Tose 于 2021 年推出的动作 RPG，以“脑力朋克”为核心概念，让 Yuito 与 Kasane 两位主角从不同路线切入同一场近未来危机。它凭借念力战斗、SAS 队友能力共享系统和极强的世界观统一性，成为 2020 年代初期很有个性的一部单作 JRPG。

**核心主题**：
- 脑力朋克世界观
- 双主角双路线叙事
- 念力动作战斗
- 连接与羁绊系统

### 2.9 《量子破碎》（2016）

- [《量子破碎》Wiki](./量子破碎_WIKI.md)

Remedy Entertainment 于 2016 年推出的第三人称科幻动作游戏，以时间机器事故为起点，把时间操控战斗、企业阴谋、宿命论和真人剧集联动结构揉成同一部作品。它凭借 `Junction` 分歧、游戏与剧集混合叙事，以及鲜明的 2010 年代跨媒体实验气质，成为一部很难被忽略的单作大企划。

**核心主题**：
- 时间操控与宿命论
- 游戏与真人剧集混合叙事
- 科技企业阴谋
- Remedy 式电影化动作实验

### 2.10 《热血无赖》（2012）

- [《热血无赖》Wiki](./热血无赖_WIKI.md)

United Front Games 于 2012 年推出的开放世界动作冒险游戏，以香港为舞台，讲述卧底警察沈威在三合会内部周旋、不断失去边界感的故事。它最令人难忘的地方，是把港式警匪片气质、拳脚格斗、街头追逐和高密度都市氛围揉成了同一部作品。虽然它开发早期与 `True Crime: Hong Kong` 有历史关联，但正式发售时已经以独立标题 `Sleeping Dogs` 上市，且后续没有发展成稳定续作体系，因此当前仍归入“未有系列”目录。

**核心主题**：
- 港式卧底警匪片
- 香港都市开放世界
- 近身格斗与环境处决
- 单作 cult classic 气质

### 2.11 《双人成行》（2021）

- [《双人成行》Wiki](./双人成行_WIKI.md)

Hazelight Studios 于 2021 年推出的合作动作冒险平台游戏，以一对走向离婚的夫妻 Cody 与 May 被迫共同冒险的设定为核心，把婚姻裂痕、家庭关系和关卡机制紧密绑在一起。它最突出的地方，是彻底围绕“双人合作”来设计整部游戏，而不是把合作当成单人游戏的附加选项。虽然它延续了 Hazelight 在《逃出生天》等作品里对合作叙事的持续探索，但本作并不属于某个正式系列，因此当前归入“未有系列”目录。

**核心主题**：
- 双人专用合作冒险
- 婚姻关系修复叙事
- 高密度机制变化
- Friend's Pass 低门槛联机

### 2.12 《逃出生天》（2018）

- [《逃出生天》Wiki](./逃出生天_WIKI.md)

Hazelight Studios 于 2018 年推出的合作动作冒险游戏，讲述 Leo 与 Vincent 两名囚犯联手越狱、一路逃亡并最终面对彼此真实立场的故事。它最关键的地方，在于把“强制双人分屏合作”直接做成了叙事和玩法的共同骨架，而不是单纯增加一个联机模式。虽然它和《双人成行》同样出自 Hazelight，也都沿着合作叙事这条路线前进，但本作并不属于正式系列，因此当前归入“未有系列”目录。

**核心主题**：
- 强制双人分屏叙事
- 越狱与亡命逃亡电影感
- 双主角关系反转
- Friend Pass 低门槛联机

### 2.13 《双影奇境》（2025）

- [《双影奇境》Wiki](./双影奇境_WIKI.md)

Hazelight Studios 于 2025 年推出的合作动作冒险游戏，围绕科幻作家 Mio 与奇幻作家 Zoe 被困入窃取创意的机器、被迫在各自故事世界中合作求生展开。它最突出的地方，是把科幻与奇幻两套风格高速切换，并继续把双人合作做成绝对前提。虽然它明显承接了《逃出生天》《双人成行》这条 Hazelight 路线，但本作并不属于某个正式系列，因此当前归入“未有系列”目录。

**核心主题**：
- 科幻与奇幻双世界合作冒险
- 友谊与创作人格碰撞
- 高密度机制切换
- Friend's Pass 与跨平台联机

### 2.14 《赛博朋克 2077》与《往日之影》（2020 / 2023）

- [《赛博朋克 2077》与《往日之影》Wiki](./赛博朋克2077_WIKI.md)

CD PROJEKT RED 于 2020 年推出的第一人称开放世界动作 RPG，以夜之城为舞台，讲述雇佣兵 V 因 Relic 生物芯片事故而被强尼·银手人格构造体侵入意识，并在公司权力、帮派秩序和个人求生之间挣扎的故事。虽然它源自 Mike Pondsmith 的《Cyberpunk》桌面 RPG 宇宙，也已经延伸出资料片《往日之影》、动画《边缘行者》和后续项目，但就当前仓库的游戏整理口径而言，主干仍集中在《赛博朋克 2077》这一部核心游戏及其唯一大型资料片，因此暂时归入“未有系列”目录。

**核心主题**：
- 赛博朋克都市与资本技术压迫
- 身体改造、人格覆盖与身份危机
- 第一人称开放世界 RPG
- 从争议首发到长期修复重估的代表性案例

## 3. 后续可收录方向

如果后续继续扩展这个目录，可以考虑收录以下类型的作品：

**国产独立单作**：
- 《光明与黑暗》系列之外的单作
- 各种尝试性作品
- 小众但有意义的作品

**国外独立单作**：
- 独立游戏佳作
- 实验性作品
- 作者型游戏

**特殊题材作品**：
- 教育/严肃游戏
- 艺术/实验游戏
- 小众题材游戏

---

## 4. 收录标准

归入"未有系列"的作品，通常满足以下条件之一：

1. 没有系列归属，是独立单作
2. 原本计划系列化，但因各种原因中断
3. 开发者只打算做一部作品
4. 题材或形式特殊，难以系列化
5. 在游戏史上有独特意义，值得单独记录

---

## 5. 一句话总结

"未有系列"收录的是那些独立存在的游戏作品，它们没有系列传承，但同样值得被铭记。
