## 1. 系列概览

《轩辕剑》是中国游戏史上最具影响力的国产 RPG 系列之一，与《仙剑奇侠传》并称"大宇双剑"。自 1990 年初代问世以来，它以"上古神话 + 历史演义 + 哲学思辨"为核心，在三十余年间塑造了无数经典角色，也留下了"壶中世界""天之痕""苍之涛"等深入人心的意象。

和《仙剑奇侠传》侧重"情感叙事"不同，《轩辕剑》的核心魅力在于它始终把"历史与哲学"放在叙事的中心位置。从初代的"人神之战"到《天之痕》的"牺牲与救赎"、《苍之涛》的"历史与命运"、《汉之云》的"大义与苍生"，每一作都在用不同的方式探讨同一个命题：在历史的洪流中，个人的选择究竟能改变什么。

如果按系列内部的开发团队与风格演变来看，《轩辕剑》大致可以分成五个阶段：

- **经典奠基阶段**：《轩辕剑》（1990）、《轩辕剑二》（1994）—— DOMO 小组的早期探索，奠定系列基础
- **外传探索阶段**：《轩辕剑外传：枫之舞》（1996）—— 系列首部外传，开创"外传品质超越正传"的传统
- **黄金巅峰阶段**：《轩辕剑三：云和山的彼端》（1999）、《轩辕剑三外传：天之痕》（2000）、《轩辕剑四：黑龙舞兮云飞扬》（2002）、《轩辕剑外传：苍之涛》（2004）—— 系列口碑与影响力的巅峰期
- **转型探索阶段**：《轩辕剑五：一剑凌云山海情》（2006）、《轩辕剑外传：汉之云》（2007）、《轩辕剑外传：云之遥》（2010）—— 3D 化转型与叙事风格的调整
- **新时代阶段**：《轩辕剑六》（2013）、《轩辕剑外传：穹之扉》（2015）、《轩辕剑柒》（2020）—— 技术革新与系列复兴的尝试

这条系列最有意思的地方，在于它既是一条游戏产品线，也是华语流行文化中少有的"历史哲学思辨载体"。从上古神话到春秋战国，从唐代欧亚旅程到三国乱世，从山海界到封神古史，《轩辕剑》始终在用游戏的方式，讲述关于"道"与"义"、"命"与"择"的故事。

## 2. 分期方式

### 2.1 经典奠基阶段

- [《轩辕剑》（1990）Wiki](./SWD_1990_WIKI.md)
- [《轩辕剑二》（1994）Wiki](./SWD_2_WIKI.md)

这一阶段是 DOMO 小组的早期探索。初代以 DOS 平台起步，奠定了"上古神话 + 历史演义"的基本框架。二代在此基础上进行了全面扩展，确立了系列的核心世界观与玩法基础。虽然以今天的眼光来看，这两作在技术与叙事上都显得稚嫩，但它们为后续的黄金时代奠定了坚实基础。

### 2.2 外传探索阶段

- [《轩辕剑外传：枫之舞》（1996）Wiki](./SWD_FENGZHISHENG_WIKI.md)

这一作是系列首部外传，以战国时期为背景，讲述了墨家弟子与机关术的故事。它在剧情深度与角色塑造上都超越了前两作，开创了"外传品质超越正传"的传统，也为后续的《天之痕》《苍之涛》树立了标杆。

### 2.3 黄金巅峰阶段

- [《轩辕剑三：云和山的彼端》（1999）Wiki](./SWD_3_WIKI.md)
- [《轩辕剑三外传：天之痕》（2000）Wiki](./SWD_3A_TIANYIHEN_WIKI.md)
- [《轩辕剑四：黑龙舞兮云飞扬》（2002）Wiki](./SWD_4_WIKI.md)
- [《轩辕剑外传：苍之涛》（2004）Wiki](./SWD_4A_CANGZHTAO_WIKI.md)

这一阶段是《轩辕剑》系列口碑与影响力的巅峰。《云和山的彼端》以唐代欧亚大陆为背景，讲述了跨越欧亚大陆的史诗旅程；《天之痕》以隋末为背景，成为系列最受欢迎的作品之一；《黑龙舞兮云飞扬》引入 3D 画面与天书系统；《苍之涛》以春秋背景探讨了历史与命运的深刻命题。这四作共同构成了《轩辕剑》最辉煌的时代。

### 2.4 转型探索阶段

- [《轩辕剑五：一剑凌云山海情》（2006）Wiki](./SWD_5_WIKI.md)
- [《轩辕剑外传：汉之云》（2007）Wiki](./SWD_5A_HANZHIYUN_WIKI.md)
- [《轩辕剑外传：云之遥》（2010）Wiki](./SWD_5B_YUNZHIYAO_WIKI.md)

这一阶段是系列的转型期。《一剑凌云山海情》以山海经为背景，引入了全新的世界观设定；《汉之云》与《云之遥》以三国时期为背景，形成了"三国三部曲"的格局。这一阶段的作品在技术上有明显提升，但也面临着玩家评价分化的挑战。

### 2.5 新时代阶段

- [《轩辕剑六》（2013）Wiki](./SWD_6_WIKI.md)
- [《轩辕剑外传：穹之扉》（2015）Wiki](./SWD_6A_QIONGZHIFEI_WIKI.md)
- [《轩辕剑柒》（2020）Wiki](./SWD_7_WIKI.md)

这一阶段是系列的技术革新期。《轩辕剑六》采用全新引擎，画面大幅提升；《穹之扉》在叙事与玩法上进行了新的尝试；《轩辕剑柒》引入即时战斗系统，是系列最大胆的革新之作。这一阶段的作品在技术层面有明显进步，但也在玩家群体中引发了关于"系列传统"与"时代革新"的讨论。

## 3. 阅读顺序

如果想按题材年代理解系列世界观，更稳妥的读法是：

1. 《轩辕剑》《轩辕剑二》：上古神话线
2. 《轩辕剑六》《轩辕剑外传：穹之扉》：商周古史线
3. 《轩辕剑外传：苍之涛》：春秋
4. 《轩辕剑外传：枫之舞》：战国
5. 《轩辕剑四：黑龙舞兮云飞扬》：秦末
6. 《轩辕剑柒》：西汉末新朝
7. 《轩辕剑外传：汉之云》《轩辕剑外传：云之遥》：三国
8. 《轩辕剑三外传：天之痕》：隋末
9. 《轩辕剑三：云和山的彼端》：唐代中后期
10. 《轩辕剑五：一剑凌云山海情》：独立的山海界故事线

## 5. 版本关系与补充说明

《轩辕剑》系列最容易混淆的地方，主要集中在"外传定位""世界观关联"和"开发团队延续"这几类关系上。

### 5.1 外传定位

系列中的"外传"作品并非简单的衍生作，而是与正传世界观紧密关联、甚至在口碑上超越正传的重要作品：

- 《轩辕剑外传：枫之舞》：系列首部外传，开创了"外传品质超越正传"的传统
- 《轩辕剑三外传：天之痕》：系列最受欢迎的作品之一，被许多玩家视为系列巅峰
- 《轩辕剑外传：苍之涛》：系列哲学思辨的巅峰之作
- 《轩辕剑外传：汉之云》与《轩辕剑外传：云之遥》：共同构成"三国三部曲"

这些外传在剧情完整性和角色塑造上都有很高评价，不应被简单视为"配角作品"。

### 5.2 世界观关联

系列世界观存在复杂的关联关系：

- **上古神话线**：《轩辕剑》《轩辕剑二》—— 以黄帝、蚩尤等上古神话为核心
- **战国线**：《轩辕剑外传：枫之舞》—— 以战国时期墨家与机关术为核心
- **商周线**：《轩辕剑六》《轩辕剑外传：穹之扉》—— 以商周之际为背景
- **秦末线**：《轩辕剑四：黑龙舞兮云飞扬》—— 以秦末为背景
- **隋唐线**：《轩辕剑三外传：天之痕》《轩辕剑三：云和山的彼端》—— 分别对应隋末与唐代
- **三国线**：《轩辕剑外传：汉之云》《轩辕剑外传：云之遥》—— 以三国时期为背景
- **山海经线**：《轩辕剑五：一剑凌云山海情》—— 以山海经世界为背景
- **西汉末新朝线**：《轩辕剑柒》—— 以王莽篡汉前后的乱世为背景

### 5.3 开发团队

系列开发团队始终是大宇资讯 DOMO 小组，但经历了多次人员变动：

- **蔡明宏**：系列制作人，被称为"轩辕剑之父"
- **郭炳宏**：早期核心成员，参与多部作品
- **吴欣叡**：编剧，负责多部作品的剧情创作

开发团队的变动直接影响了各阶段的风格差异，也是理解系列评价分化的重要背景。

### 5.4 核心设定

系列存在一些贯穿始终的核心设定：

- **轩辕剑**：上古神器，系列的核心道具，代表着"王道"与"正义"
- **壶中世界**：炼妖壶的内部世界，是系列重要的世界观设定
- **天书**：从四代开始引入的系统设定，也是剧情中的重要道具
- **机关术**：墨家机关术，在多部作品中出现
- **上古十神器**：包括轩辕剑、炼妖壶、伏羲琴、神农鼎、昆仑镜等

## 7. 一句话总结

《轩辕剑》之所以能成为华语游戏史上的文化符号，不只是因为它活得够久，而是因为它在三十余年间，始终在用游戏的方式，讲述关于"道"与"义"、"命"与"择"的故事。
