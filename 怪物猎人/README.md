# 《怪物猎人》系列 Wiki 目录

> 说明：本目录用于集中整理《怪物猎人》（Monster Hunter / モンスターハンター）系列相关 Wiki 条目。本文参考 Capcom 官方公开资料、Capcom 支持页、Wikipedia、Monster Hunter Wiki 与公开资料整理而成，写法以“中文整理型单作 Wiki + 系列索引”为主，默认包含版本关系说明与一定程度的设定、流程剧透。

---

## 1. 系列概览

《怪物猎人》是 Capcom 自 2004 年起持续扩张的“狩猎动作”系列，也是过去二十年里最能代表日本动作游戏设计演化的一条长线品牌之一。它最早并不是今天大家熟悉的那种全球级共斗大作，而是从 PlayStation 2 时代一套相当生硬、却也非常鲜明的原型出发：接任务、进地图、追踪大型怪物、采集素材、制作武器与防具，再去挑战更强的目标。后来几乎所有系列名场面，无论是四人联机狩猎、用怪物素材打造装备、为不同武器练动作节奏，还是“打一只怪就是一整场战斗”的张力，最初都能在这套原型里找到雏形。

这个系列之所以适合做“整理型 Wiki”，是因为它很容易被新读者误解成一条简单的编号正传线。实际情况并不是这样。《怪物猎人》长期同时存在编号作、`G / Ultimate` 加强版、`Portable / Freedom` 掌机重构版、区域命名差异，以及后来《World》《Rise》这种全球统一命名的新阶段。如果不先把“哪部是原版、哪部是加强版、哪部是在另一平台上重做平衡与内容结构的近亲作”理清，系列目录很快就会写散。

当前这个目录会优先按“世代原点 + 版本关系难点”来推进。第一批先从 PS2 奠基阶段写起，把初代《Monster Hunter》、其加强版《Monster Hunter G》，以及真正把系列推进到第二世代的《Monster Hunter 2（Dos）》梳理清楚。先把这三作讲明白，后面再去看《Portable / Freedom》线、`G / Ultimate` 体系和《World》《Rise》时代，就会容易很多。

## 3. 阅读顺序

如果是第一次接触《怪物猎人》系列，比较适合先按下面两层顺序理解。

第一层，是最基础的“原点阅读顺序”：

1. 《怪物猎人》初代
2. 《怪物猎人 G》
3. 《怪物猎人 Portable / Freedom》
4. 《怪物猎人 2（Dos）》
5. 《怪物猎人 Portable 2nd / Freedom 2》
6. 《怪物猎人 Portable 2nd G / Freedom Unite》
7. 《怪物猎人 Tri》
8. 《怪物猎人 Portable 3rd》
9. 《怪物猎人 3G / 3 Ultimate》
10. 《怪物猎人 4》
11. 《怪物猎人 4G / 4 Ultimate》
12. 《怪物猎人 X / XX / Generations / Generations Ultimate》

这条线的好处是很直接。你会先看到系列最原始、也最粗粝的动作和任务结构，再看到 Capcom 如何通过 `G` 版给初代补足更高阶内容与亚种体系，接着看到《Portable / Freedom》怎样把第一世代怪猎改造成掌机可扩散的形态，再理解《2（Dos）》如何把第二世代系统往深处推进，看《Portable 2nd》与《2nd G》怎样把这套东西真正推向大众，随后进入《Tri》《Portable 3rd》《3G》这组三连理解第三世代如何拆开、重做、再整合，最后再看《4》《4G》怎样把立体战斗和旅团冒险做成熟，以及《X / XX》如何把旧时代怪猎做成一场庆典式总结。

第二层，如果后续继续扩写，比较自然的系列阅读顺序通常会是：

1. 《怪物猎人》
2. 《怪物猎人 G》
3. 《怪物猎人 Portable / Freedom》
4. 《怪物猎人 2（Dos）》
5. 《怪物猎人 Portable 2nd / Freedom 2》
6. 《怪物猎人 Portable 2nd G / Freedom Unite》
7. 《怪物猎人 Tri》
8. 《怪物猎人 Portable 3rd》
9. 《怪物猎人 3G / 3 Ultimate》
10. 《怪物猎人 4》
11. 《怪物猎人 4G / 4 Ultimate》
12. 《怪物猎人 X / Generations》
13. 《怪物猎人 XX / Generations Ultimate》
14. 《Monster Hunter: World》
15. 《Monster Hunter World: Iceborne》
16. 《Monster Hunter Rise》
17. 《Monster Hunter Rise: Sunbreak》
18. 《Monster Hunter Wilds》

这里需要特别注意的是，《怪物猎人》并不是一个主要靠连续剧情驱动的系列，所以“阅读顺序”更多是在帮读者理解系统演化、平台迁移和版本关系，而不是在追一条不能跳开的主线故事。

## 5. 版本关系与补充说明

《怪物猎人》系列最容易混淆的地方，不只是编号，而是“原版、G 版、Portable 版、Ultimate 版”经常被混成一个概念。

当前最需要先记住的几条关系包括：

- 2004 年 PS2《Monster Hunter》是系列起点，也是“狩猎动作”这一套循环真正落地的原型。
- 2005 年 PS2《Monster Hunter G》不是全新编号续作，而是建立在初代之上的加强版，核心是扩充任务层级、加入更多高阶内容和亚种体系。
- 2009 年 Wii《Monster Hunter G》不是另一部新作，而是对 `G` 版的再发行与功能调整版本；它与 PS2 版主体内容接近，但官方支持页明确说明加入了按钮攻击等若干改良。
- PSP《Monster Hunter Portable / Monster Hunter Freedom》虽然和《Monster Hunter G》内容关系很深，但它不是单纯把 `G` 原样搬上掌机，而是掌机化之后重新组织出来的分支入口。
- 2006 年《Monster Hunter 2（Dos）》是第二部编号正作，也是第二世代起点；它和初代、`G` 同属 PS2 语境，但不是“G 的再加强版”。
- 《Monster Hunter Portable / Freedom》与《Monster Hunter G》内容关系很深，但它不是 `G` 的直接移植，而是掌机化重构版。
- 官方支持页还特别说明，《Monster Hunter 2（Dos）》不能直接继承《Monster Hunter》《Monster Hunter G》或《Monster Hunter Portable》的角色存档；这也说明它虽然承接前作经验，却仍是重新起步的一代。
- 《Monster Hunter Portable 2nd / Freedom 2》虽然大量吸收《Dos》内容，但它不是 `Dos` 原样搬上 PSP，而是更适合掌机节奏的再组织版本。
- 《Monster Hunter Portable 2nd G / Freedom Unite》则是建立在《Portable 2nd / Freedom 2》上的大型加强版，不是独立编号正作。
- `Freedom`、`Freedom 2`、`Freedom Unite` 是海外命名；日本本土对应的是 `Portable`、`Portable 2nd`、`Portable 2nd G`。
- 《Monster Hunter Tri》是第三部编号正作，而《Portable 3rd》不是《Tri》的掌机移植版，而是以第三世代内容为基础重组出的掌机新作。
- 《Portable 3rd》没有正式欧美本地化版本，因此不存在严格对应的官方 `Freedom 3` 商品名。
- 《Monster Hunter 3G / 3 Ultimate》是建立在《Tri》基础上的大型强化版，同时回收了《Portable 3rd》阶段的部分第三世代内容；它不是 Wii《Tri》的简单高清复刻。
- 3DS《3G》和 Wii U《HD Ver.》之间可以共享存档，但 Wii《Tri》与《3G》之间没有继承关系。
- 《Monster Hunter 4》原版只在日本单独发售；海外玩家主流接触到的第四世代起点通常其实是《4 Ultimate》。
- 《Monster Hunter 4G / 4 Ultimate》是建立在《4》基础上的大型强化版，不应和《X / XX》视作同一种“后续作品”。
- 《Monster Hunter X / Generations》更接近第四世代末期的纪念型重组作，而《XX / Generations Ultimate》则是其强化完整版。
- 《Generations Ultimate》在今天常被当作旧结构《怪物猎人》的重要完整入口之一，但它代表的是庆典式汇总路线，不是编号正作主线推进。
- 早期《怪物猎人》的线上服务已经全部结束，因此今天讨论这些作品时，需要区分“历史上的完整联网生态”和“如今还能实际游玩的离线内容”。

也就是说，理解这个系列最好的方式，不是简单记“哪作最强”，而是先分清：

- 原版正作
- `G / Ultimate` 加强版
- `Portable / Freedom` 掌机化重构版
- 编号续作与世代转换作
- 线上功能仍在不在，以及这会怎样影响今天回看这些作品
