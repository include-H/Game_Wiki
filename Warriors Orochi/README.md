# 《无双大蛇》系列 Wiki 目录

> 说明：本目录用于集中整理《无双大蛇》（無双OROCHI / Warriors Orochi）系列 Wiki 条目。本文主要依据 KOEI TECMO / GAMECITY 官方资料、Wikipedia 与相关公开 Wiki 整理而成，默认涉及系列分期、版本关系、人物势力与一定剧情剧透。当前只讨论《无双大蛇》系列本身，不把《真三国无双》与《战国无双》正传目录混写，也不并入《无双全明星》或其他跨系列联动作品。

---

## 1. 系列概览

《无双大蛇》是 Omega Force 以《真三国无双》与《战国无双》两条主线为基础，重新拼接出来的跨系列动作分支。它最特别的地方，不只是把中国三国与日本战国的人物放到同一张角色表里，而是直接创造了一个被“远吕智”扭曲过的异世界，让原本属于不同历史、不同叙事语法的人物，被迫在同一个战场逻辑里重新结盟、对立、甚至重写关系。

如果说《真三国无双》强调的是“三国乱世里的群雄战场”，《战国无双》强调的是“日本战国人物的个人传奇”，那么《无双大蛇》真正有意思的地方，就是它既不是前两者的简单拼盘，也不是随便找个理由让大家打一架。它更像是一条独立长出来的“神魔化无双”支线：先用远吕智把时空捏碎，再让历史英雄被卷进神话危机，最后一步步把世界观越做越夸张。

这个系列的早期发展尤其清楚。初代《无双OROCHI》负责证明“三国 + 战国”的大乱炖真的能成立；《无双OROCHI 魔王再临》则把神仙、妖魔、补完剧情和角色规模整个抬上去，形成真正意义上的强化续篇；《无双OROCHI Z》进一步把前两作整合到 PS3 世代，补足新角色与新剧本，成为早期《无双大蛇》最典型的一次“完全版阶段总结”。后面《无双OROCHI 2》《3》《4》虽然继续扩张，但早期这三步已经把系列的基础气质立得很清楚。

因此，阅读《无双大蛇》时，最好不要把它简单当成《真三国无双》外传去看。它和本家共享角色资源与基础动作语法，但它的世界观、剧情推进、版本关系和历史定位都已经足够独立，值得被当成一条单独的系列线来整理。

## 2. 分期方式

### 2.1 早期异世界三部曲

- [《无双OROCHI》Wiki](./WARRIORS_OROCHI_1_WIKI.md)
- [《无双OROCHI 魔王再临》Wiki](./WARRIORS_OROCHI_MAOU_SAIRIN_WIKI.md)
- [《无双OROCHI Z》Wiki](./WARRIORS_OROCHI_Z_WIKI.md)

这一阶段是《无双大蛇》真正把自己定义清楚的时期。初代先用“远吕智扭曲时空”把跨系列框架搭起来；《魔王再临》把故事补到更大，也把神话与仙界成分全面抬高；《Z》则把前两作整合并升级成 PS3 时代的完整阶段版本。

### 2.2 编号化扩张期

- [《无双OROCHI 2》Wiki](./WARRIORS_OROCHI_2_WIKI.md)
- [《无双OROCHI 3》Wiki](./WARRIORS_OROCHI_3_WIKI.md)

这一阶段是系列编号化、规模化、神魔化最明显的一轮扩张。《无双OROCHI 2》把故事推进到“妖蛇毁世 + 时间回溯救场”的规模，并围绕 `Special / Hyper / Ultimate` 长出一整条加强版链；《无双OROCHI 3》则进一步引入更高层神系力量，把世界观推向超大型神话 crossover，而 `Ultimate` 基本上也成了这一代的标准完全版。

### 2.3 现代再整合期

- [《无双OROCHI 4》Wiki](./WARRIORS_OROCHI_4_WIKI.md)
- [《无双OROCHI 4 Ultimate》Wiki](./WARRIORS_OROCHI_4_ULTIMATE_WIKI.md)

这一阶段重点是系列如何在新一轮主机世代里继续延展神话规模，同时回头收束旧角色资源与新系统结构。《无双OROCHI 4》负责把系列重新拉回“纯无双战场动作 + 神术系统 + 奥林匹斯诸神”的大框架；《无双OROCHI 4 Ultimate》则通过新角色、追加剧情、Infinity / Challenge 等内容，把这一阶段补成真正意义上的完全版。

## 3. 阅读顺序

比较推荐按下面顺序阅读：

1. 《无双OROCHI》
2. 《无双OROCHI 魔王再临》
3. 《无双OROCHI Z》
4. 《无双OROCHI 2》
5. 《无双OROCHI 3》
6. 《无双OROCHI 4》
7. 《无双OROCHI 4 Ultimate》

现在已经可以一路从初代读到《无双OROCHI 4 Ultimate》。前半段三作构成系列如何成立、扩张并被《Z》整合总结的早期链条；中段 2 与 3 两作把这条线真正推成大型编号分支；4 与 `4 Ultimate` 则展示了系列在现代平台上如何继续扩大神话规模，同时再次通过“本体 + 完全版”结构完成收束。

如果你只是想先抓“早期无双大蛇最完整的版本感”，那么《无双OROCHI Z》仍然是最直接的入口；如果你更想看系列在现代平台上最完整、最方便接触的神话大乱战形态，那么《无双OROCHI 4 Ultimate》会更像终点式入口。

## 5. 版本关系与补充说明

- 《无双大蛇》是独立系列，不应并入《真三国无双》或《战国无双》正传目录。它虽然共用大量角色与动作基础，但世界观和剧情主线都另起了一条分支。
- 日文《無双OROCHI 魔王再臨》在欧美通常被称为 `Warriors Orochi 2`。也就是说，这一作在海外承担的是“系列第二作编号位”，而不是单纯副标题扩展。阅读外文资料时必须特别留意这一点。
- 《无双OROCHI Z》虽然没有用阿拉伯数字编号，但它并不是可以一句带过的移植合集。它把初代《无双OROCHI》和《魔王再临》内容整合到 PS3 / PC 平台，并加入额外角色与新剧本，事实上是早期系列最重要的阶段整合版之一。
- 《Special》《Hyper》《Ultimate》《Z》这类名称，在《无双大蛇》系列里往往不只是换皮再版，而常常意味着角色、剧本、平台与完整体验层面的明显差异。后续写编号化阶段时，这一点会越来越重要。
- 《无双OROCHI 2》本体最早对应 PS3 / Xbox 360，而《Special》《Hyper》《Ultimate》分别承担掌机强化、Wii U 强化与真正意义上的综合完全版职责。讨论这代时，不能把它们全都笼统视为“移植版”。
- 《无双OROCHI 3》在海外名称上对应 `WARRIORS OROCHI 4`，而《无双OROCHI 3 Ultimate》则是这一代最重要的后续强化版本。Steam 上的 `WARRIORS OROCHI 3 Ultimate Definitive Edition` 本质上是 `Ultimate` 的进一步整合发行形式。
- 《无双OROCHI 4》在海外名称上对应 `WARRIORS OROCHI 4`，日文语境里的数字会比欧美少 1。它首发时已经是纯关卡制回归加神术 / 神器系统，而不是延续开放世界或其他本家实验方向。
- 《无双OROCHI 4 Ultimate》不是单纯年度版，它有独立标题、完整发售版本、`Upgrade Pack` 升级路线，以及对主线疑问的补完作用。讨论现代阶段最完整体验时，通常应以 `Ultimate` 为准。
- 目前条目会优先把对理解“完整体验”有明显意义的扩展版本并入正文说明；纯服装、纯特典、纯数字附赠内容则只做简要带过。

---

## 6. 一句话总结

《无双大蛇》真正特别的地方，不是它把不同系列角色塞到一起，而是它把“历史无双”硬生生扭成了一条带神魔、异界和时空错乱色彩的独立动作史诗支线。
