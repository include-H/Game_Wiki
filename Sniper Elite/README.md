# 《狙击精英》系列 Wiki 目录

> 说明：本目录用于集中整理《狙击精英》（Sniper Elite）系列相关 Wiki 条目。这个系列最早从 2005 年的柏林潜入射击起步，后来逐步发展成 Rebellion 最稳定、辨识度也最高的二战战术狙击品牌之一。它的核心并不是单纯“打一枪看击杀特写”，而是把远距离弹道、地形观察、噪音掩护、潜入渗透与半开放任务结构，慢慢打磨成了一整套很有自己味道的设计语言。

---

## 1. 系列概览

《狙击精英》表面上看是一条二战第三人称射击系列，但它真正留下印象的地方，是把“狙击手如何在战场上耐心观察、换位、等待噪音掩护、利用弹道修正完成任务”做成了主循环。初代作品已经有风偏、下坠、心跳和伪装指数这些要素；到了后续作品，系列又把 X-Ray Kill Cam、多人合作、目标猎杀和大地图渗透不断强化，逐渐从“有特色的冷门二战射击”长成“提到狙击潜入就绕不开”的固定招牌。

如果按今天回头看，这个系列大致可以分成三段：

- 柏林起点阶段：以 2005 年初代《Sniper Elite》和 2012 年《Sniper Elite V2》为核心，确立 Karl Fairburne、柏林终局、OSS 潜入与弹道狙击的系列母题。
- 北非到意大利扩张阶段：以《Sniper Elite 3》《Sniper Elite 4》为核心，把舞台从柏林拉向北非和意大利，并把关卡做得更开放、更强调路线选择与任务拼接。
- 法国成熟阶段：从《Sniper Elite 5》到 2025 年的《Sniper Elite: Resistance》，系列把“开放战区 + 潜入狙击 + 支线目标 + 可重玩暗杀”这套结构进一步稳定下来。

需要额外说明的是，《Zombie Army》虽然和《狙击精英》在技术和角色层面有直接血缘关系，但它已经发展成独立衍生系列；《Sniper Elite VR》则属于围绕系列核心手感做出的 VR 分支，不等同于主线编号续作。

## 3. 阅读顺序

如果是第一次接触这个系列，最自然的阅读顺序仍然建议按作品发售顺序来读，因为这样最容易看清它是如何一步步长成今天这个样子的：

1. 《Sniper Elite》（2005）
2. 《Sniper Elite V2》
3. 《Sniper Elite 3》
4. 《Sniper Elite 4》
5. 《Sniper Elite 5》
6. 《Sniper Elite: Resistance》

如果只是想快速理解“现代《狙击精英》为什么成立”，也可以先读《Sniper Elite V2》，再接《3》和《4》。因为从 V2 开始，系列的招牌元素已经比较完整：

- Karl Fairburne 作为固定主角
- 柏林 / 北非 / 意大利这些更强调任务渗透的舞台
- X-Ray Kill Cam 带来的强烈视觉记忆点
- 合作模式与重复游玩价值逐步增强

## 4. 当前已覆盖条目

目前这个目录已经覆盖《狙击精英》从起点到“现代成熟前夜”的关键四作：

- 《Sniper Elite》（2005）
- 《Sniper Elite V2》与《Sniper Elite V2 Remastered》
- 《Sniper Elite 3》
- 《Sniper Elite 4》

这四篇条目合起来，已经可以把系列最重要的一条主脉络梳理清楚：

- Karl Fairburne 作为德国出生、为 OSS 服务的美军狙击手，是如何被固定下来的
- 柏林终局为什么会在初代和 V2 里被反复重写
- 系列为何会从线性狙击关卡，逐步长成更开放的战术渗透射击
- 《Target Hitler》《Save Churchill》《Target Führer》《Deathstorm》这类 DLC 是怎样把“历史人物猎杀”和额外战役扩展成系列固定组件的

## 5. 版本关系与补充说明

- 《Sniper Elite V2》不是单纯高清移植版，而是在初代同一大战背景与主角框架上做出的现代续作 / 重写版本；它和初代同样都发生在 1945 年柏林，但叙事目标和关卡结构并不相同。
- 《Sniper Elite V2 Remastered》是 2019 年推出的复刻版，整合原版 DLC，并加入画面升级、照片模式和更多可操作角色；理解 V2 时，通常最好把原版与复刻版放在一起看。
- 《Sniper Elite 3 Ultimate Edition》与《Sniper Elite 4 Deluxe Edition》本质上都属于“本体 + 关键 DLC”的整合包装，其中《Save Churchill》与《Deathstorm》因为补充了完整支线战役，在阅读和游玩体验上都值得单独点名。
- 《Sniper Elite: Nazi Zombie Army》从 V2 的技术与玩法框架中分化出来，后来演变成独立的《Zombie Army》系列。它和《狙击精英》血缘很深，但不应直接算作主线编号续作。
- 《Sniper Elite VR》采用新主角与 VR 第一人称形态，更适合视作围绕系列手感展开的独立旁支。

---

## 6. 一句话总结

《狙击精英》系列真正特别的地方，不只是“能打出很夸张的慢镜头”，而是它把二战狙击、潜入、弹道与开放关卡慢慢熬成了一套越来越成熟的战术射击语法。
