## 1. 系列概览

《地平线》（Horizon）是 Guerrilla Games 为 PlayStation 时代打造的新原创科幻动作 RPG 系列。它最鲜明的辨识度，在于把“部落文明”“远未来废墟”与“巨型机械兽生态”这三种看似不太相容的元素缝成了一个完整世界：人类像重新回到原始社会，但旧世界的人工智能、自动化系统和生态灾难并没有真正离开，只是换了一种方式继续支配大地。

如果按当前最常见的阅读和理解方式来看，这个系列大致可以先分成两条主脉：

- Aloy 的主线成长与旧世界真相
- 以机械生态、部族文化与失控 AI 为核心的世界观扩展

## 3. 推荐阅读顺序

如果是第一次接触《地平线》系列，最适合先从下面顺序开始：

1. 《地平线：零之曙光》
2. 《地平线：零之曙光 冰尘雪野》
3. 《地平线：西之绝境》
4. 《地平线：西之绝境 炽焰海岸》

需要说明的是：

- 《零之曙光》是理解整个系列世界观、旧文明毁灭原因与 Aloy 身份的绝对起点。
- 《冰尘雪野》虽然是扩展内容，但对 Banuk 文化、机械异常与后续叙事气质都有补充意义。
- 如果后续目录继续扩写，也可以单独拆出角色、部族、AI 系统和机械兽专题页。
