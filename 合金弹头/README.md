> 说明：本目录用于集中整理《合金弹头》（Metal Slug）系列相关 Wiki 条目。本文主要依据 SNK 官方页面、Steam 商店页、Wikipedia、Metal Slug Wiki 与公开资料整理而成，默认包含系列剧情、版本关系与一定剧透说明。

## 1. 系列概览

《合金弹头》是街机时代最有代表性的 2D 横版动作射击系列之一，也是 SNK 旗下辨识度最高的长线品牌之一。它表面上看是“跑、跳、开枪、上载具、炸 Boss”的爽快街机游戏，真正让它在玩家记忆里站稳的，却是那套既夸张又细腻的战争喜剧审美：叛军、木乃伊、火星人、古代遗迹、实验兵器和各种奇怪战车，全都能被塞进同一个战场里，而且居然成立。

如果把整个系列按历史阶段来理解，大致可以分成四段：

- 街机黄金期：`Metal Slug`、`Metal Slug 2 / X`、`Metal Slug 3`
- SNK 重组后的延续期：`Metal Slug 4`、`Metal Slug 5`、`Metal Slug 6`
- 掌机 / 家用机分支期：`1st Mission`、`2nd Mission`、`Advance`、`3D`、`7`、`XX`
- 手机与新世代衍生期：`Defense`、`Attack`、`Commander`、`Awakening`、`Attack Reloaded`、`Tactics`

这个系列最特殊的地方，在于它虽然有连续世界观和 recurring 角色，但并不依赖长篇对白来推动叙事。很多剧情是用关卡演出、敌军构成、地图异变和结尾反转完成的。因此写《合金弹头》Wiki，不能只抄一句“打败摩登将军”就结束，而是要把每一作战役到底发生了什么、敌人为何变化、系列世界观如何扩张写清楚。

## 3. 阅读方式

如果想按系列发展来读，最推荐直接从总条目进入。那篇主文已经把：

- 全系列主要作品按时间顺序串起来
- 各作基础信息、平台与定位整理在同一篇里
- 每一作剧情单独拆成小节展开
- 玩法、角色、势力和历史定位统一归纳

也就是说，这个目录当前不是“每作单独一篇”的拆分结构，而是先以一篇总览文把《合金弹头》整个系列打通。

## 4. 一句话总结

《合金弹头》最厉害的地方，不只是街机手感，而是它把战争、荒诞、喜剧和灾难电影气质压进了同一套像素横版动作语法里。
