## 1. 系列概览

《红色警戒》（Red Alert）是《命令与征服》宇宙中最具大众知名度的分支之一，也是许多中文玩家最熟悉的即时战略系列。它从“爱因斯坦回到过去刺杀希特勒，导致苏联与盟军在另一条历史线上爆发大战”这一极具辨识度的架空历史设定起步，随后逐渐发展出更夸张、更娱乐化，也更具流行文化传播力的战争科幻风格。

如果按当前已经整理和准备整理的内容来看，这个系列大致可以分成三个核心阶段：

- 初代阶段：《红色警戒》与资料片《反戈一击》《劫后余生》
- 黄金时期：《红色警戒 2》与《尤里的复仇》
- 后期转向：《红色警戒 3》与独立资料片《起义时刻》

这套系列最特别的地方，不只是阵营鲜明、单位夸张或 FMV 演出抓眼，而是它始终把大战略题材拍得既像冷战幻想，又像某种带有强烈娱乐感的平行世界战争秀。特斯拉线圈、天启坦克、光棱科技、尤里的心灵控制、旭日帝国的变形机甲，这些元素早就超出了传统“写实战争 RTS”的范围，也因此形成了《红警》极强的独立辨识度。

## 3. 推荐阅读顺序

如果想按发售顺序理解系列发展，比较适合这样读：

1. 《红色警戒》
2. 《红色警戒 2》
3. 《尤里的复仇》
4. 《红色警戒 3》
5. 《起义时刻》

如果想按系列气质变化来读，也可以这样理解：

- 想看系列如何从架空冷战和 Westwood 经典 RTS 起步，就从初代开始。
- 想看《红警》最具大众记忆度、阵营设计和单位魅力都最强的一段，就重点读《红警 2》和《尤里的复仇》。
- 想看系列如何进一步夸张化、海战强化和三阵营化，就读《红警 3》与《起义时刻》。

## 4. 版本关系说明

《红色警戒》系列最容易混淆的地方，主要集中在资料片与后来的重制 / 再发行关系上：

- 初代《Command & Conquer: Red Alert》有两个官方资料片：`Counterstrike` 与 `The Aftermath`
- 2020 年推出的 `Command & Conquer Remastered Collection` 收录了《Command & Conquer》与《Red Alert》以及全部 3 个资料片，其中《Red Alert》部分自然也包含 `Counterstrike` 和 `The Aftermath`
- 《Red Alert 2》与资料片《Yuri's Revenge》现在通常以一个打包条目理解，Steam 也将两者作为统一产品再发行
- 《Red Alert 3: Uprising》并不是传统意义上必须依赖本体启动的资料片，而是单独发行的 standalone expansion

也就是说，讨论时最好先区分：

- 本体
- 官方资料片
- 后来的合集 / 重制版
- 真正独立运行的扩展作品
