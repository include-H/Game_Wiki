# 《镜之边缘》系列 Wiki 目录

> 说明：本目录用于集中整理《镜之边缘》（Mirror's Edge）系列相关 Wiki 条目。本文主要依据 EA 官方页面、Steam 商店页、Wikipedia 与 Mirror's Edge Wiki 等公开资料整理而成，默认包含剧情概述、版本关系与一定剧透说明。

---

## 1. 系列概览

《镜之边缘》系列最鲜明的识别度，不是“未来都市”这几个字本身，而是它把第一人称视角、跑酷移动、极简色彩和高空失重感捏成了一整套几乎一眼就能认出来的游戏语言。很多游戏都会让玩家奔跑、攀爬和跨越障碍，但《镜之边缘》真正做出来的，是一种“身体先于枪械、路线先于火力、速度先于停留”的游玩逻辑。

初代《Mirror's Edge》于 2008 年推出，背景是一个由高压秩序、信息控制和监控体系维持表面洁净的城市。玩家扮演跑者 Faith，在屋顶、空调外机、吊臂和玻璃幕墙之间穿梭，把本来多用于背景装饰的城市立面，变成了真正的行动空间。它在当年并不是最传统意义上的“大众爆款”，却迅速成了第一人称跑酷题材的代表作。

如果从系列结构来看，《镜之边缘》目前真正构成主干阅读路线的核心作品只有两部：2008 年的初代《Mirror's Edge》，以及 2016 年的《Mirror's Edge Catalyst》。两者都以 Faith 为中心，也都发生在高度控制的未来都市，但它们并不是简单的前后剧情承接关系。`Catalyst` 更接近一次重构或重新诠释：保留了系列气质与核心角色名号，同时重写了世界观基底、角色关系和叙事起点。

这个系列体量不大，却很适合做“整理型 Wiki”。原因在于它的开发信息、版本关系和作品定位非常容易被混淆。很多中文讨论会把 `Catalyst` 直接说成“镜之边缘 2”，但从官方命名和剧情设定看，它并不是编号续作；而初代虽然没有大型剧情资料片，却有 `Pure Time Trials Map Pack` 这种和核心玩法高度一致的 DLC，也有 iOS / 手机版等衍生版本。若不先把这些关系讲清，系列目录就很容易失焦。

从历史位置看，《镜之边缘》更像一个“影响力大于体量”的系列。它未必像某些年货级大作那样靠数量堆出庞大宇宙，但它在视觉设计、路线引导、第一人称移动表达和跑酷题材市场认知上留下的印记非常深。后来很多作品谈到第一人称移动、美术极简与城市疾走感时，几乎都会被拿来和它比较。

## 3. 阅读顺序

如果想按系列最自然的理解顺序阅读，建议按下面的次序：

1. 《Mirror's Edge》（2008）
2. 《Mirror's Edge》相关补充：`Pure Time Trials Map Pack`
3. 《Mirror's Edge Catalyst》（2016）

需要特别说明的是：

- 《Mirror's Edge Catalyst》不是初代剧情的直接后续，而是重构作。
- 因此最合适的读法，不是把它理解成“Faith 故事线性延续的第二章”，而是把它看成“同一品牌核心概念在新世代硬件上的重新搭建”。
- 如果读者只想先理解系列为什么重要，那么从初代开始仍然是最稳妥的入口；如果更在意现代操作手感和较大地图结构，再回头看 `Catalyst` 也成立。

## 4. 当前已覆盖条目

目前本目录已覆盖《镜之边缘》系列两部最核心的主机 / PC 主干作品：

- 《Mirror's Edge》（2008）
- 《Mirror's Edge Catalyst》（2016）

这两篇条目合起来，已经能把系列最关键的脉络说明清楚：

- 初代如何用极简色彩、屋顶疾走和第一人称视角建立题材辨识度
- `Pure Time Trials` 这类内容为何更像对核心跑酷设计的补充，而不是改写主线定位的大型扩展
- `Catalyst` 为什么更接近一次世界与人物关系的重构，而不是传统意义上的《镜之边缘 2》

## 5. 版本关系与补充说明

- 《Mirror's Edge》最早于 2008 年登陆 PlayStation 3 与 Xbox 360，2009 年登陆 Windows；移动端后来也出现过不同形式的衍生版本，但主干讨论仍以主机 / PC 本体为准。
- 初代最需要一并提到的补充内容是 `Pure Time Trials Map Pack`。它没有明显扩写主线剧情，却很能体现系列把“跑酷路线本身”做成挑战内容的设计取向。
- 2008 年初代与 2016 年 `Catalyst` 虽然都以 Faith 为主角、都发生在高压控制的未来都市，但两者不是严格连续的单一剧情时间线。
- 《Mirror's Edge Catalyst》采用重新搭建的世界观与角色关系，因此写作时应把它视作重构作，而不是默认套用初代的组织设定、人物经历与历史背景。
- `Catalyst` 的在线功能后来已结束服务，今天回看这部作品时，更应把重点放在本体主线、开放区域跑酷和整体重构思路上，而不是把它理解成长期运营型在线作品。

---

## 6. 一句话总结

《镜之边缘》系列最难被替代的地方，是它把“跑起来”这件事本身，做成了世界观、视觉设计和游戏手感共同成立的核心表达。
