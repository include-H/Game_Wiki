## 1. 系列概览

“Tom Clancy's” 在游戏领域并不是单一作品名，而更像一条长期使用的军事惊悚品牌线。它通常围绕情报战、反恐行动、特种作战、现代军事冲突和国家安全议题展开，不同子系列虽然玩法差异很大，但大多共享一种相对严肃、偏现实主义的现代表达框架。

在 Ubisoft 长期经营下，这个品牌逐步扩展出多条风格差异明显的游戏路线。最典型的几支包括：

- 《细胞分裂》：以 Sam Fisher 为核心的潜行谍报路线
- 《彩虹六号》：以反恐突击和战术行动为核心的路线
- 《幽灵行动》：以特种部队与现代战场协同为核心的路线
- 《鹰击长空》：以现代空战为核心的路线
- 《终结战争》：以近未来大战略与指挥为核心的路线
- 《全境封锁》：虽然更偏在线化和后期作品形态，但同样属于 Ubisoft 旗下的 Tom Clancy's 品牌线

这些作品并不是共享同一条连续剧情宇宙的“编号正传”，更适合被理解为同一品牌下的多个子系列。因此，后续目录也会优先采用“父目录统筹 + 子系列 README + 单作 Wiki”的方式来整理，而不是把所有作品混成一个大目录。

## 4. 版本关系与补充说明

“Tom Clancy's” 品牌下最容易混淆的地方，不是某一作的副标题，而是不同子系列本来就不该混为同一条主线。

当前需要先分清的几件事包括：

- 《细胞分裂》《彩虹六号》《幽灵行动》等是同一品牌下的不同子系列，不是互相编号续作
- 同一品牌下不同系列的开发团队、玩法结构与叙事重点可能完全不同
- 真正需要做“发售顺序阅读”的，往往是子系列内部，而不是整个 Tom Clancy's 品牌横向混读
- 因此目录结构上更合理的做法，是先建父目录，再让每个子系列拥有自己的 README 与单作页
