## 1. 系列概览

《彩虹六号》是 Tom Clancy's 品牌下最具代表性的战术反恐系列之一，也是“先规划、再突入”的小队战术射击传统里最有影响力的一条线。和后来很多把“战术”理解为技能、英雄或枪械配装的多人射击不同，早期《彩虹六号》真正的核心，在于行动前规划路线、切分小队、设置爆破点、决定进门时机，然后在极短时间内把一次高风险人质营救或反恐突击执行干净。

系列从 1998 年初代《Tom Clancy's Rainbow Six》起步，最早的气质非常硬，也非常像小说式反恐行动模拟。到了《Rogue Spear》，它把这套多小队规划和全球反恐任务继续做大做厚；再到《Rainbow Six 3: Raven Shield》，系列在视觉、手感和现代 3D 表达上完成明显升级，也成为很多玩家心中“经典老《彩虹六号》”最有代表性的样子。

如果按今天比较适合理解的阶段感来看，《彩虹六号》大致可以先分成四段：

- 经典规划式反恐阶段：`Rainbow Six`、`Rogue Spear`、`Raven Shield`
- 主机化与近距离战术阶段：`Lockdown`、`Vegas`、`Vegas 2`
- 重启实验阶段：`Patriots`（未成形）到 `Siege` 之前的思路变化
- 现代运营阶段：`Rainbow Six Siege`

这条系列最有意思的地方，在于它的早期和后期几乎像两个时代的不同答案。早期更像反恐战术模拟，强调行动计划、路线控制和一击成败；后来的《Siege》则把战术拆进了角色能力、地图破坏和长期对抗运营里。因此，如果不先把经典三作看清，很容易低估《彩虹六号》最初其实有多“慢”、多“硬”、也多讲究计划。

## 5. 版本关系与补充说明

《彩虹六号》系列最容易混淆的地方，主要集中在“资料片 / 任务包关系”“PC 经典时期与主机时期的表达差异”，以及“经典规划式反恐”和后来《Siege》完全不是同一种玩法结构”这几类问题上。

当前最需要先记住的几条包括：

- 初代《Rainbow Six》最重要的扩展内容是 `Eagle Watch`，它是经典早期体验的一部分。
- 《Rogue Spear》后续的重要扩展包括 `Urban Operations` 与 `Covert Ops Essentials`，讨论这部作品时最好一起看。
- 《Raven Shield》最常见的完整讨论，也会把 `Athena Sword` 与 `Iron Wrath` 一并纳入版本关系说明。
- 《Lockdown》更适合被理解为转型前奏，而不是继续延长经典三作原有规划式结构的正统直线续写。
- 《Vegas》与《Vegas 2》属于同一条明显主机化、掩体化、近距离突击表达路线，讨论它们时不应再套用经典 PC 规划阶段的标准。
- 《Siege》虽然仍然叫《彩虹六号》，但它的长期运营属性、干员体系与可破坏环境逻辑已经属于另一种玩法结构，讨论时最好和经典期、《Vegas》期分别处理。
- 早期《彩虹六号》强调的是任务规划、路径和时机控制，不能把后来的《Vegas》或《Siege》那种节奏反套回去。

也就是说，讨论《彩虹六号》时最好先区分：

- 经典 PC 规划式战术阶段
- 主机化近距离战术阶段
- 现代多人运营阶段
- 各代资料片与任务扩展的整合阅读关系
