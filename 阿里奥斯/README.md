# 《阿里奥斯》游戏 Wiki 目录

> 说明：本目录用于集中整理大连阿里奥斯科技咨询有限公司（以及公开资料中与 `softzen` 标识相关的同系作品）在中文手机游戏时代推出的相关条目。本文主要依据公开百科资料、公开怀旧站点与转载页面整理而成。当前先以《时光八音盒》与其后继《时光樱花册》这条恋爱叙事线为起点，后续再继续补充同厂其他作品。

---

## 1. 系列概览

如果说奥尔资讯那批作品更像华语 Java 恋爱 AVG 对偶像剧、校园幻想与萌系题材的横向试探，那么阿里奥斯这条线给人的感觉，则更偏向“连续剧式恋爱故事的阶段推进”。《时光八音盒》本身就是一个分为三部的连续结构，而《时光樱花册》又明确被公开资料视作它的续作。这使得它比很多同时代手机恋爱游戏更适合被当成一条连续作品线来整理。

这一系作品最值得先立父目录的地方，在于它不是简单的“几款名字相似的恋爱手游”，而是确实存在前后承接：先有《时光八音盒》三部曲，再有《时光樱花册》继续延伸。对于 Java 手游时代来说，这种持续连载式、并且有明确续接关系的恋爱叙事并不算常见。

从现有公开资料看，这条线的整体气质仍然属于中文手机恋爱 / AVG 语境，但比起单部完结的小体量作品，它更强调章节推进、情感延续和前后代的故事传承感。这也是当前目录先从它入手的原因。

## 3. 阅读顺序

推荐按下面顺序阅读：

1. 《时光八音盒》
2. 《时光樱花册》

其中《时光八音盒》内部又建议按三部曲顺序理解：

1. 《时光八音盒1：萌之篇》
2. 《时光八音盒2：忆之篇》
3. 《时光八音盒3：梦之节》

如果再接《时光樱花册》，则可以把它视作这条线从三部曲走向后继新篇章的自然延伸。

## 4. 版本关系与补充说明

这一父目录当前最容易混淆的地方，不是哪一部有没有高清重制，而是“《时光八音盒》是单作还是三部曲合集，以及《时光樱花册》到底和它是什么关系”。

当前可先分清的几件事包括：

- 《时光八音盒》不是单一一部，而是由《萌之篇》《忆之篇》《梦之节》构成的三部曲
- 《时光樱花册》在公开资料里通常被直接视作《时光八音盒》的续作
- 《时光樱花册》至少可见《遗失之心》与《泪之背影》两部相关标题
- 现存公开资料主要围绕 Java 时代版本与后来的转载下载页，未见明确的大型 DLC / 资料片体系
