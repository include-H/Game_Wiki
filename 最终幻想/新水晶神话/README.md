# 《最终幻想》新水晶神话目录

> 说明：本目录用于单独整理 `Fabula Nova Crystallis Final Fantasy`，也就是中文常见所说的“新水晶神话”相关条目。之所以单列，是因为这一块最容易和普通编号正传关系混淆：它们共享神话框架与企划脉络，但并不等于同一部作品拆章节。

---

## 1. 目录定位

“新水晶神话”并不是一个玩家随口起的别名，而是 Square Enix 曾经正式提出的一套共享神话企划框架。最核心的理解方式，不是把它当成“又一个连续宇宙”，而是把它看成一组共享底层神话名词和创作母题的作品群。

当前这个子目录主要覆盖下面几条主线：

- [《最终幻想 XIII》及其衍生 / 后续作品](./FINAL_FANTASY_XIII_AND_SEQUELS_WIKI.md)
- [《最终幻想零式》](./FINAL_FANTASY_TYPE_0_WIKI.md)
- [《Final Fantasy Versus XIII》与《最终幻想 XV》Wiki](../FINAL_FANTASY_VERSUS_XIII_AND_XV_WIKI.md)

---

## 2. 当前已覆盖

当前目录已覆盖：

1. 《最终幻想 XIII》及其衍生作品
2. 《最终幻想零式》
3. 《Final Fantasy Versus XIII》到《最终幻想 XV》的企划转化说明

其中《最终幻想 XV》虽然最终已脱离“新水晶神话”名义独立发售，但若不先理解 `Versus XIII` 这层前史，就很难读清它今天的结构与世界观表达；因此这一页仍作为本目录的重要关联入口。

---

## 3. 一句话总结

“新水晶神话”不是《最终幻想》里最容易懂的一块，但恰恰因为它牵涉企划变形、共享神话和多部作品分流，所以更需要单独立目录慢慢理清。
