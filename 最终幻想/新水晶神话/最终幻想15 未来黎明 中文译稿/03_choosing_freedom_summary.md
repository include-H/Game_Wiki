# 《选择自由》简介

《选择自由》是《最终幻想XV：未来黎明》的第三章，主轴落在 Lunafreya 死后复生后的觉醒、动摇与自我选择。故事开篇承接 Altissia 之后的空白，Luna 从海底般的沉眠中醒来，发现自己竟在地下墓所中重新获得了生命。她很快意识到，世界早已不是自己记忆中的模样：天空被永夜笼罩，Noctis 已失踪十年，而自己也早被世人认定为亡者。更令人不安的是，她体内多出了一种能够吸收 daemon 病祸的力量，这让她既能击退怪物，也逐渐朝“非人之物”滑去。

在逃亡途中，Luna 遇见了年轻猎人 Sol。两人最初彼此提防，却在一路同行、并肩战斗中慢慢建立起信任。Sol 代表的是永夜时代挣扎求生的人类视角：她不理解 Oracle 对诸神与使命的绝对服从，也不断逼问 Luna“你自己究竟想要什么”。这种追问动摇了 Luna 过去从未怀疑过的信念。她在梦中先后得到 Bahamut 与 Gentiana 的启示，也由此发现真相并不单纯：Bahamut 赋予她新生命，并非只是为了拯救世界，而是想借她收集全部黑暗，为彻底毁灭被视作“病田”的 Eos 做准备。与此同时，Luna 还从初代 Oracle Aera 的异象中看见了 Ardyn 作为救世者的过去，第一次真正理解了他为何会对诸神与命运怀有如此深重的怨恨。

后半章中，Luna 与 Sol 前往 Ralmuell 遗迹救援 Aranea 小队，并合力击败 Sapphire Weapon。Luna 更进一步以自己的身体吸收病祸，成功将已经魔物化的 Aranea 救回人形，却也因此让自己彻底显露出近似 daemon 的外貌，随即被 Aranea误伤、遭到 Glaive 与猎人们拘押送回 Lestallum。此时，Sol 终于明确站到 Luna 一边，帮助她逃走；而 Luna 也在 Gentiana 的最后警告中下定决心，不再只是神意的傀儡，而要亲自选择未来。第三章结尾，Luna 孤身重返 Insomnia，在王座之间面对 Ardyn，没有直接宣战，而是提出一句出人意料的话：“我是否能请求你助我一臂之力？”这标志着她已经从单纯执行使命的 Oracle，迈向真正以自身意志行动的“Luna”。
