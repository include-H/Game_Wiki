# 《炼金工房》伊莉丝系列 Wiki 目录

> 说明：本目录用于整理《炼金工房》在格拉姆纳特之后的“伊莉丝”阶段，也就是通常所说的《伊莉丝的炼金工房 永恒玛那》《伊莉丝的炼金工房2 命运贤者》《伊莉丝的炼金工房3 伟大幻象》三作。本文主要依据 Atelier 25 周年官方页面、Atelier Wiki、Wikipedia 与公开资料整理而成，写法以“中文整理型单作 Wiki + 系列索引”为主。

---

## 1. 系列概览

伊莉丝系列在整个《炼金工房》史里很特殊，因为它是品牌第一次明显往“更传统的冒险 RPG”方向倾斜的阶段。前面的萨尔布鲁克和格拉姆纳特，虽然也有探索、战斗和剧情推进，但骨架始终还是围绕工房、调和、时限和日常经营展开。到了《伊莉丝》，Gust 明显开始尝试另一种展开方式：更强调主线推进、更强调迷宫和战斗、更强调一条相对明确的冒险叙事。

这并不意味着《炼金工房》突然变成了别的系列，而更像是它在 PS2 中期市场环境里做出的一次主动换挡。炼金术仍然很重要，素材和道具循环也还在，但玩家的整体感受已经更接近“带着炼金系统的王道 JRPG”，而不是“以工房生活为中心的慢节奏成长记”。

三作内部的定位也并不一样：

- 《伊莉丝的炼金工房 永恒玛那》负责把系列真正推向主线冒险 RPG 化
- 《伊莉丝的炼金工房2 命运贤者》虽然发售在后，但时间位置更接近前传，重点是把阿尔文等人与魔法文明冲突写得更完整
- 《伊莉丝的炼金工房3 伟大幻象》则继续保留战斗和探索驱动，但整体结构更偏任务与异界迷宫，和前两作关系也没那么紧密

也正因为如此，伊莉丝三作虽然通常被并称为一个阶段，但阅读时最好同时注意“发售顺序”和“世界观前后位置”并不完全一致。

## 3. 阅读顺序

如果按发售顺序理解伊莉丝系列的演化，推荐这样读：

1. 《伊莉丝的炼金工房 永恒玛那》
2. 《伊莉丝的炼金工房2 命运贤者》
3. 《伊莉丝的炼金工房3 伟大幻象》

这是最适合看清系列风格变化的顺序，因为你能直接看到《炼金工房》怎样一步步把战斗、迷宫和主线叙事抬到更前面。

如果更重视世界观内部时间位置，可以先把《伊莉丝的炼金工房2 命运贤者》理解成更靠前的故事，再看《永恒玛那》。但对大多数读者来说，仍然建议优先按发售顺序读，因为《永恒玛那》承担的是“这一阶段先把方向改给玩家看”的作用。

《伊莉丝的炼金工房3 伟大幻象》则最适合放在最后，因为它虽然继承了这一阶段的战斗化倾向，却已经不像前两作那样主要服务于同一条人物与世界关系线。

## 4. 版本关系与补充说明

伊莉丝系列最需要先分清的，主要是下面几条关系：

- 《伊莉丝的炼金工房 永恒玛那》于 2004 年在 PlayStation 2 首发，是《炼金工房》品牌第一次明显以传统冒险 RPG 面貌进入欧美市场的作品
- 《伊莉丝的炼金工房2 命运贤者》于 2005 年首发，虽然发售在《永恒玛那》之后，但其时间位置通常被理解为更偏前传或前史补足
- 《伊莉丝的炼金工房3 伟大幻象》于 2006 年首发，保留 RPG 化骨架，但整体结构与前两作的连续感已明显减弱
- 这三作都没有今天意义上的大型剧情 DLC，理解重点主要在本体平台发行与不同地区版本时间差
- 它们在系列史上的最大意义，不是版本花样多，而是共同构成了《炼金工房》最早、也最明确的一次“主线冒险 RPG 化”阶段

## 6. 一句话总结

伊莉丝系列最关键的历史价值，是它让《炼金工房》第一次大幅度往“更像传统冒险 RPG”的方向迈了一步。
