> 说明：本目录用于集中整理 Gust / 光荣特库摩旗下《炼金工房》（Atelier）系列相关 Wiki 条目。本文主要依据 Atelier 25 周年官方页面、各作公开产品页、Wikipedia 与 Atelier Wiki 等公开资料整理而成，写法以“中文整理型单作 Wiki + 系列索引”为主，默认包含剧情概述、版本关系与一定剧透说明。当前先采用父子文件夹结构处理，因为《炼金工房》本身更适合按子系列而不是按单一编号主线来理解。

## 1. 系列概览

《炼金工房》是日本 RPG 里非常独特的一条长寿系列。它不像很多传统 JRPG 那样总把“拯救世界”放在最前面，而是长期把炼金术、委托、采集、调和、经营小工房和角色日常生活放在叙事中心。很多作品当然也会碰到国家危机、世界异变或更大的命题，但真正让这个系列稳定成立的，一直是那种“从小地方做起，把配方、素材、委托和伙伴关系一点点经营起来”的节奏。

这个系列之所以适合单独建父目录，是因为它内部并不是一条从 1 代一路直推到今天的连续编号主线。更常见也更稳妥的理解方式，是把它视作多个子系列依次展开：

- 萨尔布鲁克系列：系列原点，确立“工房经营 + 时间限制 + 小规模目标”的基本语法
- 格拉姆纳特系列：把双主角、地点切换和阶段性目标继续往前推
- 伊莉丝 / 玛那学园阶段：更明显地往传统 JRPG 主线和战斗驱动靠拢
- 亚兰德系列：重新把生活感、期限感和现代角色魅力集中到系列中心
- 黄昏系列：把炼金术重新放回一个正在慢慢衰退、却仍必须认真生活的世界里
- 不可思议系列：把炼金术重新写回“发现、想象与探索本身就值得高兴”的状态
- 秘密系列：把《炼金工房》做成了一条真正有现代连续成长感的主角三部曲

换句话说，《炼金工房》真正难整理的地方，不是作品数量多，而是每一代到底该放进哪条子系列、哪些作品共享世界观、哪些只是共享系列玩法传统。也正因为如此，这个父目录更适合先把“怎么分支”说清楚，再让各个子系列自己处理阅读顺序、版本关系和单作条目。

## 2. 分期方式

结合官方周年整理与当前目录规划，《炼金工房》父目录暂时先按下面方式理解：

### 2.1 萨尔布鲁克系列

- [《萨尔布鲁克》子系列目录](./萨尔布鲁克系列/README.md)

这是整个系列的原点，也是“玛莉 - 艾莉 - 莉莉”这条最早主干所在的阶段。它最重要的价值，不只是时间最早，而是很多后来会被视作《炼金工房》标志的东西，例如工房生活、期限压力、委托成长和地区性冒险规模，都已经在这里成型。

### 2.2 格拉姆纳特系列

- [《格拉姆纳特》子系列目录](./格拉姆纳特系列/README.md)

这一阶段用《优蒂》《薇欧蕾特》把系列从萨尔布鲁克带到一片新大陆，也让“工房经营”进一步转成“新世界安顿”与“村镇振兴”这两种更明确的目标结构。

### 2.3 伊莉丝系列

- [《伊莉丝》子系列目录](./伊莉丝系列/README.md)

这一阶段用《永恒玛那》《命运贤者》《伟大幻象》把《炼金工房》第一次大幅推向更传统的冒险 RPG 轨道，也因此成为整个系列风格变化最明显的一段。

### 2.4 玛那学园系列

- [《玛那学园》子系列目录](./玛那学园系列/README.md)

这一阶段把《伊莉丝》已经做强的 RPG 化方向重新装回学院结构之中，也因此形成了一个气质很鲜明的“校园炼金 RPG”分支。

### 2.5 亚兰德系列

- [《亚兰德》子系列目录](./亚兰德系列/README.md)

这一阶段把《炼金工房》的生活感、轻盈感和现代玩家最熟悉的人物魅力重新集中起来，也因此成为系列面向更广泛玩家的重要入口之一。

### 2.6 黄昏系列

- [《黄昏》子系列目录](./黄昏系列/README.md)

这一阶段把《炼金工房》的生活感放进了一个正在衰退的世界里，也因此写出了系列里最有环境重量、但仍然不失温度的一条线。

### 2.7 不可思议系列

- [《不可思议》子系列目录](./不可思议系列/README.md)

这一阶段把《炼金工房》重新拉回“发现新事物本身就很快乐”的状态，也因此成为系列里最有童话感与探索欲的一条线。

### 2.8 秘密系列

- [《秘密》子系列目录](./秘密系列/README.md)

这一阶段把《炼金工房》真正推进到更广泛的现代全球玩家视野里，也让系列少见地拥有了一条连续成长感极强的主角三部曲。

### 2.9 后续待补子系列

- 《蕾斯莱莉娅娜的炼金工房》等近年移动 / 跨平台扩展线

当前父目录先把结构搭好，后续再按子系列逐步扩展。

## 3. 阅读顺序

如果想按《炼金工房》系列整体演化来理解，比较稳的阅读顺序是：

1. 先看 [《萨尔布鲁克》子系列目录](./萨尔布鲁克系列/README.md)，理解系列原点
2. 再看 [《格拉姆纳特》子系列目录](./格拉姆纳特系列/README.md)，理解早期新阶段如何展开
3. 再看 [《伊莉丝》子系列目录](./伊莉丝系列/README.md)，理解系列第一次明显 RPG 化的阶段
4. 再看 [《玛那学园》子系列目录](./玛那学园系列/README.md)，理解学院 RPG 化这一段如何成立
5. 再看 [《亚兰德》子系列目录](./亚兰德系列/README.md)，理解系列怎样在现代语境下重新找回生活感与人气
6. 再看 [《黄昏》子系列目录](./黄昏系列/README.md)，理解系列怎样把生活感重新压进衰退世界
7. 再看 [《不可思议》子系列目录](./不可思议系列/README.md)，理解系列怎样把发现感和炼金本身的神奇感重新立起来
8. 再看 [《秘密》子系列目录](./秘密系列/README.md)，理解系列怎样在现代语境下把连续成长三部曲真正做成大众入口
9. 再按后续补档进度，逐步看《蕾斯莱莉娅娜的炼金工房》等扩展线

如果只看当前已经覆盖的内容，推荐先按萨尔布鲁克系列的发售顺序阅读：

1. 《玛莉的炼金工房：萨尔布鲁克的炼金术士》
2. 《艾莉的炼金工房：萨尔布鲁克的炼金术士2》
3. 《莉莉的炼金工房：萨尔布鲁克的炼金术士3》

这个顺序最适合看清三件事：

- 系列最早是如何把炼金术写成“生活职业”而不是“大战斗系统”的
- 续作怎样把原本较小的工房循环扩成更完整的校园、委托和同伴结构
- 第三作怎样回到前史位置，把萨尔布鲁克王国与学院建立的背景补起来

如果想继续往下接，下一步最合适的顺序是：

4. 《优蒂的炼金工房：格拉姆纳特的炼金术士》
5. 《薇欧蕾特的炼金工房：格拉姆纳特的炼金术士2》

这两作最适合接在萨尔布鲁克之后，因为它们一方面保留了早期《炼金工房》的生活与时限传统，另一方面又已经明显开始把舞台和目标结构写得更开。

如果继续往后接，下一步推荐这样读：

6. 《伊莉丝的炼金工房 永恒玛那》
7. 《伊莉丝的炼金工房2 命运贤者》
8. 《伊莉丝的炼金工房3 伟大幻象》

这一段最适合放在格拉姆纳特之后，因为它清楚展示了《炼金工房》从“生活系工房 RPG”往“更传统的冒险 RPG”转向的过程。

如果继续往后接，下一步推荐这样读：

9. 《玛那学园 炼金术士们的学园》
10. 《玛那学园2 衰退的炼金术士们》

这两作适合放在《伊莉丝》之后，因为它们没有把 RPG 化方向退回去，而是把这套骨架重新装进了学院生活与青春群像结构里。

如果继续往后接，下一步推荐这样读：

11. 《罗罗娜的炼金工房 亚兰德的炼金术士》
12. 《托托莉的炼金工房 亚兰德的炼金术士2》
13. 《梅露露的炼金工房 亚兰德的炼金术士3》
14. 《露露亚的炼金工房 亚兰德的炼金术士4》

这四作适合放在《玛那学园》之后，因为它们清楚展示了《炼金工房》如何把生活感、期限感、旅行感和现代角色魅力重新集中起来。

如果继续往后接，下一步推荐这样读：

15. 《爱夏的炼金工房 黄昏大地之炼金术士》
16. 《艾斯卡与罗吉的炼金工房 黄昏天空之炼金术士》
17. 《夏莉的炼金工房 黄昏海洋之炼金术士》

这三作适合放在亚兰德之后，因为它们把《炼金工房》的生活感重新放进了一片逐渐衰退的世界中，也因此形成了风格反差非常鲜明的一段。

如果继续往后接，下一步推荐这样读：

18. 《苏菲的炼金工房 不可思议书的炼金术士》
19. 《苏菲的炼金工房2 不可思议梦的炼金术士》
20. 《菲莉丝的炼金工房 不可思议旅的炼金术士》
21. 《莉迪＆苏瑞的炼金工房 不可思议绘画的炼金术士》

如果按发售顺序理解，也可以把《苏菲2》放到最后。但如果更重视人物关系和时间线连贯性，这个顺序会更顺。

如果继续往后接，下一步推荐这样读：

22. 《莱莎的炼金工房 常暗女王与秘密藏身处》
23. 《莱莎的炼金工房2 失落传说与秘密妖精》
24. 《莱莎的炼金工房3 终结之炼金术士与秘密钥匙》

这三作适合放在不可思议之后，因为它们几乎代表了《炼金工房》在现代全球语境下最完整、也最容易被连续追下去的一条主角成长线。

## 5. 版本关系与补充说明

《炼金工房》父目录最容易混淆的地方，不是 DLC，而是“哪些作品属于同一子系列、哪些只是同属上位品牌”。

当前需要先分清的几件事包括：

- 《炼金工房》是上位系列名，内部长期按不同世界观和不同角色组分成多个子系列
- 早期的萨尔布鲁克、格拉姆纳特、伊莉丝、玛那学园和后来的亚兰德、黄昏、不可思议、秘密，彼此不等于单一连续主线
- 因此目录结构上，更合理的做法是先建《炼金工房》父目录，再让各子系列拥有自己的 README 与单作页
- 《玛莉的炼金工房》在 2023 年推出了《玛莉的炼金工房 Remake ～萨尔布鲁克的炼金术士～》，这让系列原点第一次以较现代、也更易接触的形式重新进入全球市场
- 但《艾莉的炼金工房》和《莉莉的炼金工房》截至 2026 年 3 月 23 日仍没有像《玛莉》那样的现代全重制全球版本，因此萨尔布鲁克三作今天的可及性并不对称
- 系列里确实存在服装包、BGM 包、便利道具等追加内容，但对早期子系列的理解而言，更重要的仍然是“本体版本差异”和“是否有重制 / Plus / 整合版”

也就是说，今后继续扩这个父目录时，核心思路仍然应该是：

- 先分子系列
- 再分作品顺序
- 最后再处理每一作自己的平台与版本关系

## 7. 一句话总结

《炼金工房》真正特别的地方，不是它做了多少炼金术设定，而是它把“做道具、接委托、经营日常与慢慢成长”写成了一整条持续近三十年的 RPG 传统。
