# 《炼金工房》黄昏系列 Wiki 目录

> 说明：本目录用于整理《炼金工房》在亚兰德之后最具世界观辨识度的“黄昏”系列，也就是通常所说的《爱夏》《艾斯卡与罗吉》《夏莉》三作。本文主要依据 Atelier 25 周年官方页面、Atelier Wiki、Wikipedia、Koei Tecmo 公开产品页与相关商店页面整理而成，写法以“中文整理型单作 Wiki + 系列索引”为主。

---

## 1. 系列概览

黄昏系列在整个《炼金工房》史上的位置非常特别，因为它几乎是系列里最明确、也最稳定的一次“衰退世界观”书写。前面的亚兰德虽然也有期限和成长压力，但整体气质始终偏明亮、轻盈、可亲近；到了黄昏，Gust 没有把这种生活感完全丢掉，却把它放进了一片资源衰竭、文明退潮、旧技术失传的世界里。

也正因为如此，黄昏三作真正动人的地方，从来不只是“世界快不行了”。更关键的是，在这样一个正在慢慢失去活力的世界里，炼金术重新变得像一门与生活、交通、粮食、医疗和地区存续都有关系的实际技术。它不只是主角的特长，也是一种对抗衰退、延缓失去的方式。

三作内部的定位也非常清楚：

- 《爱夏的炼金工房》负责把黄昏世界真正立起来，也是这一阶段情绪最温柔、也最忧伤的入口
- 《艾斯卡与罗吉的炼金工房》把视角转向边境开发与官办项目，用双主角结构让黄昏世界变得更具体
- 《夏莉的炼金工房》则把“黄昏”的长期后果进一步推到前台，同时取消时限，形成一种更自由但也更接近终段的收束感

也就是说，黄昏系列并不是单纯靠压抑取胜，而是在一整条逐步下沉的世界线上，持续讨论“人还能不能在变坏的时代里认真生活”。

## 3. 阅读顺序

如果按发售顺序理解黄昏系列，推荐这样读：

1. 《爱夏的炼金工房 黄昏大地之炼金术士》
2. 《艾斯卡与罗吉的炼金工房 黄昏天空之炼金术士》
3. 《夏莉的炼金工房 黄昏海洋之炼金术士》

这也是最推荐的理解方式。原因很简单：

- 《爱夏》负责先让玩家理解“黄昏”到底意味着什么
- 《艾斯卡与罗吉》在此基础上把世界运行和边境开发写得更立体
- 《夏莉》则进一步把衰退后果和收束感推向前景

和亚兰德系列一样，黄昏三作基本就是一条很顺的发售顺序阅读线。

## 4. 版本关系与补充说明

黄昏系列最需要先分清的，主要是下面几条关系：

- 《爱夏的炼金工房》原版首发于 2012 年 PlayStation 3，后有 `Plus` 版与 2018 年 `DX` 版
- 《艾斯卡与罗吉的炼金工房》原版首发于 2013 年 PlayStation 3，后有 `Plus` 版与 2019 年 `DX` 版
- 《夏莉的炼金工房》原版首发于 2014 年 PlayStation 3，后有 `Plus` 版与 2019 年 `DX` 版
- 对今天的现代玩家来说，讨论黄昏三作时，若不特别说明，通常默认都是 `DX` 语境
- 三作都存在服装、BGM 或附加角色等内容，但对理解这条线来说，最关键的仍然是 `Plus / DX` 整合与现代入口，而不是零散 DLC

## 6. 一句话总结

黄昏系列最动人的地方，是它没有把末世写成纯粹绝望，而是把“认真生活、认真做事”本身写成了对抗衰退的方式。
