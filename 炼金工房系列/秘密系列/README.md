# 《炼金工房》秘密系列 Wiki 目录

> 说明：本目录用于整理《炼金工房》在现代全球范围内知名度最高的“秘密”系列，也就是通常所说的《莱莎》三部曲。本文主要依据 Atelier 25 周年官方页面、Atelier Wiki、Wikipedia、Koei Tecmo 公开产品页与相关商店页面整理而成，写法以“中文整理型单作 Wiki + 系列索引”为主。

---

## 1. 系列概览

秘密系列之所以在整个《炼金工房》史上格外重要，不只是因为它卖得更好，也不只是因为莱莎成了系列最出圈的角色之一。更关键的是，它把《炼金工房》长久以来那些很难同时兼顾的东西，重新做成了一套更容易被现代玩家理解的组合：明亮的夏日冒险感、可读性更强的调和系统、更顺滑的探索和战斗节奏，以及一条真正连续推进、角色会明显一起长大的三部曲主线。

如果说亚兰德是把系列重新拉回现代玩家视野的重要阶段，那么秘密系列则更像是把《炼金工房》真正推进到更大规模全球受众面前的一段。它既保留了系列最核心的“慢慢做东西、慢慢变强、和伙伴一起把日子过成冒险”这套气质，又在视觉表达、系统入口和叙事连贯性上变得更直接。

三作内部的定位也很明确：

- 《莱莎的炼金工房 常暗女王与秘密藏身处》负责把这一条线真正立起来，也是整段故事的夏日起点
- 《莱莎的炼金工房2 失落传说与秘密妖精》把时间推到几年后，写出明显的成年感与回返感
- 《莱莎的炼金工房3 终结之炼金术士与秘密钥匙》则承担三部曲真正的收束任务，把世界规模、角色关系和系统自由度都推到最高

也正因为如此，秘密系列最适合被当成一条真正连续的现代三部曲来读，而不是几部共享主角脸谱的松散关联作。

## 3. 阅读顺序

如果按发售顺序理解秘密系列，推荐这样读：

1. 《莱莎的炼金工房 常暗女王与秘密藏身处》
2. 《莱莎的炼金工房2 失落传说与秘密妖精》
3. 《莱莎的炼金工房3 终结之炼金术士与秘密钥匙》

这也是最推荐的理解方式。因为秘密系列最重要的价值之一，就在于你能明显看见：

- 少年时代的“想去外面看看”如何变成真正的冒险
- 几年后的莱莎和伙伴们怎样带着成长与分离感重新相聚
- 三代如何把这条线真正收拢成一段完整的青春 / 成长三部曲

秘密系列没有像不可思议那样复杂的“后发售但时间在中间”的情况，因此按发售顺序读就是最稳的路径。

## 4. 版本关系与补充说明

秘密系列最需要先分清的，主要是下面几条关系：

- 三作都存在标准版、`Digital Deluxe Edition`、季票或更高整合包等版本差异，但最关键的仍然是本体主线
- 《莱莎的炼金工房》与《莱莎2》常见的重要追加内容，主要集中在服装、地图、额外区域和季票包；它们会补体验，但不至于改写主线骨架
- 《莱莎3》的版本差异相对更受关注，因为它承担三部曲收束位置；但对大多数读者来说，本体主线依然是理解这条线最核心的内容
- 这三作不存在像《罗罗娜》那样“原版 / 新版 / DX”多层重整关系，因此版本理解相对直接
- 对今天的现代玩家来说，秘密系列也是整个《炼金工房》里最容易直接从现行平台进入的一条线
