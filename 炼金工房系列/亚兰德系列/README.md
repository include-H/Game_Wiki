# 《炼金工房》亚兰德系列 Wiki 目录

> 说明：本目录用于整理《炼金工房》在 PS3 / PS Vita 到现代复刻阶段最具代表性的“亚兰德”系列，也就是通常所说的《罗罗娜》《托托莉》《梅露露》《露露亚》四作。本文主要依据 Atelier 25 周年官方页面、Atelier Wiki、Wikipedia、Koei Tecmo 公开产品页与相关商店页面整理而成，写法以“中文整理型单作 Wiki + 系列索引”为主。

---

## 1. 系列概览

亚兰德系列在整个《炼金工房》史上的地位非常关键，因为它几乎承担了“把系列重新拉回大众视野”的任务。前面的《伊莉丝》和《玛那学园》已经把《炼金工房》明显推向更传统的 RPG 骨架，但到了亚兰德，Gust 又把重心重新调了回来：战斗和冒险当然还在，但“接委托、做道具、和朋友来往、在轻盈节奏里慢慢成长”的那种系列感，重新回到了前台。

这也是为什么很多现代玩家一提到《炼金工房》的经典气质，第一反应不是更早的萨尔布鲁克，也不是《伊莉丝》式战斗化，而是亚兰德这一段。它把早期系列的生活感、期限感、道具循环和人物关系，重新包进了更现代、也更鲜明的人设、插画和平台发行环境里。

四作内部的定位也很清楚：

- 《罗罗娜的炼金工房》负责把亚兰德这条线真正立起来
- 《托托莉的炼金工房》把视野从王都和工房推向远方，是这条线最典型的“出门旅行”一作
- 《梅露露的炼金工房》把目标进一步转成开发与建设，让亚兰德三部曲形成收束感
- 《露露亚的炼金工房》则是多年之后回到旧舞台的正统后续作，承担怀旧与再连接的双重功能

也正因为如此，亚兰德系列非常适合按一条子系列完整整理：它既共享世界与角色谱系，也有非常清楚的代际推进感。

## 3. 阅读顺序

如果按发售顺序理解亚兰德系列，推荐这样读：

1. 《罗罗娜的炼金工房 亚兰德的炼金术士》
2. 《托托莉的炼金工房 亚兰德的炼金术士2》
3. 《梅露露的炼金工房 亚兰德的炼金术士3》
4. 《露露亚的炼金工房 亚兰德的炼金术士4》

这也是最推荐的理解路径。因为亚兰德系列的魅力，很大程度上就在于你能看见：

- 罗罗娜如何把这个世界先立起来
- 托托莉怎样把舞台真正推开
- 梅露露如何让个人成长和国家建设叠在一起
- 露露亚怎样在多年后把旧角色与旧舞台重新串起来

和《伊莉丝》那种“时间位置需要额外说明”的阶段不同，亚兰德系列基本就是一条非常顺的发售顺序阅读线。

## 4. 版本关系与补充说明

亚兰德系列最需要先分清的，主要是下面几条关系：

- 《罗罗娜的炼金工房》原版首发于 2009 年 PlayStation 3，后有 `Plus` 路线和 2013 年《新·罗罗娜的炼金工房 起始的故事 ～亚兰德的炼金术士～》，再到 2018 年 `DX` 版
- 《托托莉的炼金工房》和《梅露露的炼金工房》都存在 PS3 原版、`Plus` 路线与 2018 年 `DX` 版。今天大多数现代玩家默认接触的是 `DX`
- 《露露亚的炼金工房》是 2019 年推出的正统后续作，不是单纯复刻，也不是旧三部曲合集附带内容
- 亚兰德系列的 DLC 与附加内容确实不少，但重要程度并不一样：对《罗罗娜》《托托莉》《梅露露》来说，最关键的是 `Plus / 新版 / DX` 之间的内容整合；对《露露亚》来说，更需要分清角色、服装和季票类追加内容与本体主线的关系
- 这一系列在现代语境下最稳的入口，通常就是《亚兰德的炼金术士 1-2-3 DX》与《露露亚的炼金工房》

## 6. 一句话总结

亚兰德系列最重要的历史价值，是它把《炼金工房》的生活感、轻盈感和人物魅力重新集中起来，并因此成为很多现代玩家真正记住这个系列的起点。
