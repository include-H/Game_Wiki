# 《炼金工房》萨尔布鲁克系列 Wiki 目录

> 说明：本目录用于整理《炼金工房》最早的“萨尔布鲁克”阶段，也就是通常所说的《玛莉》《艾莉》《莉莉》三作主干。本文主要依据《玛莉的炼金工房 Remake》官方页面、Atelier 25 周年官方页面、Wikipedia 与 Atelier Wiki 等公开资料整理而成，写法以“中文整理型单作 Wiki + 系列索引”为主。

---

## 1. 系列概览

如果说后来的《炼金工房》已经形成了“子系列轮换”的长寿品牌结构，那么萨尔布鲁克系列代表的，就是一切刚刚长出自己的时候。它没有后来那么多花样繁复的调和系统，也没有再往后那些更明显面向现代市场的角色包装，但它最关键的东西都已经在了：开一家工房、在有限时间里学习炼金术、接委托、交朋友、提升评价，并在一个不算宏大的地区舞台里完成属于自己的成长。

萨尔布鲁克三作之所以重要，不只是因为它们最早，而是因为它们把《炼金工房》从一开始就和传统“勇者打魔王”式 RPG 拉开了距离。《玛莉》更像是毕业考试与生活再建，《艾莉》把学院和伙伴结构扩得更完整，《莉莉》则进一步把时间位置往前拨，去补足王立学院和萨尔布鲁克国家层面的背景。这三作放在一起看，能非常清楚地看见系列从“小工房经营”走向“有历史感的地区炼金传统”。

## 3. 阅读顺序

如果想按作品诞生顺序理解萨尔布鲁克系列的演化，推荐这样读：

1. 《玛莉的炼金工房：萨尔布鲁克的炼金术士》
2. 《艾莉的炼金工房：萨尔布鲁克的炼金术士2》
3. 《莉莉的炼金工房：萨尔布鲁克的炼金术士3》

这个顺序最适合看清系列最早的系统扩张过程：从玛莉的个人毕业课题，到艾莉把学院、伙伴和事件做得更丰富，再到莉莉回头补足学院建立前史与王国层面的发展需要。

如果想按世界观时间位置理解，通常更适合这样看：

1. 《莉莉的炼金工房：萨尔布鲁克的炼金术士3》
2. 《玛莉的炼金工房：萨尔布鲁克的炼金术士》
3. 《艾莉的炼金工房：萨尔布鲁克的炼金术士2》

因为《莉莉》更像一部前史补完作，而《艾莉》则明确接在《玛莉》之后，承担直接续作的位置。

## 4. 版本关系与补充说明

萨尔布鲁克系列最需要先分清的，主要是下面几条关系：

- 《玛莉的炼金工房》是系列原点。它后续有 `Plus`、PC 版、Dreamcast / PS2 收录版与 2023 年完整重制版《玛莉的炼金工房 Remake ～萨尔布鲁克的炼金术士～》
- 《艾莉的炼金工房》是接在《玛莉》之后的正统续作，也有 `The Salburg Collection` 与《萨尔布鲁克的炼金术士1・2》这类合集收录关系，但截至 2026 年 3 月 23 日仍无现代全球重制版
- 《莉莉的炼金工房》首发于 PlayStation 2，本身就比前两作晚一代硬件；其 `Plus` 版属于内容强化路线，但它没有像《玛莉》那样在现代平台获得全球全重制
- 萨尔布鲁克三作几乎没有今天意义上的“大型剧情 DLC”概念，因此理解它们时，重点不是 DLC，而是 `Plus`、合集收录和是否重制
- 《莉莉》在时间位置上更接近前传，因此它虽然发售最晚，却不适合机械地当成《艾莉》之后的单纯剧情延续

## 6. 一句话总结

萨尔布鲁克系列最值得回头看的地方，是它从一开始就把“拯救世界”换成了“经营自己的人生和工房”，而这一点后来几乎定义了整个《炼金工房》。
