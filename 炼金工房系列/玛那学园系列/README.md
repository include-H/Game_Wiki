# 《炼金工房》玛那学园系列 Wiki 目录

> 说明：本目录用于整理《炼金工房》在《伊莉丝》之后的“玛那学园”（Mana Khemia）阶段，也就是通常所说的《玛那学园 炼金术士们的学园》《玛那学园2 衰退的炼金术士们》两作。本文主要依据 Atelier 25 周年官方页面、Atelier Wiki、Wikipedia 与公开资料整理而成，写法以“中文整理型单作 Wiki + 系列索引”为主。

---

## 1. 系列概览

玛那学园系列在整个《炼金工房》史里的位置很微妙，也很有意思。它一方面明显延续了《伊莉丝》阶段已经建立起来的 RPG 化方向，战斗、迷宫、装备和主线推进依然很重要；但另一方面，它又没有继续把重心放在古代文明、世界危机或一路远行的冒险感上，而是把舞台重新收束到“学校”这个结构里。

也正因为如此，玛那学园两作给人的感觉很特别。它们既不像萨尔布鲁克和格拉姆纳特那样是工房生活主导，也不像《伊莉丝》那样不断把冲突压到宏大旅途和失落文明上。它们更像是把《炼金工房》的炼金术、RPG 战斗和校园群像揉到一起，形成一种更青春、更章节化、也更强调同学关系和学期节奏的表达。

这一阶段最值得单独建目录的原因，也就在这里：

- 它不是《伊莉丝》的简单别名
- 它也不是回到旧式工房经营
- 它更像《炼金工房》在 PS2 后期做出的一次“学院 RPG 化”分支

所以理解玛那学园系列时，最重要的不是先问它像不像传统《炼金工房》，而是先看它如何把系列里已经越来越明显的战斗与叙事驱动，重新塞回一个学园日常和成长节奏里。

## 3. 阅读顺序

如果按发售顺序理解玛那学园系列，推荐这样读：

1. 《玛那学园 炼金术士们的学园》
2. 《玛那学园2 衰退的炼金术士们》

这个顺序也是最推荐的理解方式。原因很简单：

- 初代负责把“学院 + 炼金 + 战斗 RPG”这套结构真正立起来
- 二代则在此基础上把角色线和学园问题继续扩写

和《伊莉丝》阶段相比，玛那学园并没有那么强的“世界观前史错位”问题，所以按发售顺序阅读基本就是最稳的方式。

## 4. 版本关系与补充说明

玛那学园系列最需要先分清的，主要是下面几条关系：

- 《玛那学园 炼金术士们的学园》于 2007 年在 PlayStation 2 首发，后推出 PSP 强化版 `Mana Khemia: Student Alliance`
- 《玛那学园2 衰退的炼金术士们》于 2008 年在 PlayStation 2 首发，后推出 PSP 版《玛那学园2 携带版+》
- 这两作都没有今天意义上的大型剧情 DLC，理解重点主要在 PS2 原版与 PSP 强化 / 携带版差异
- 它们在系列史上的核心意义，不是平台花样，而是把《炼金工房》进一步转成“学期推进 + 同学关系 + 校园事件”的结构
- 这一阶段通常会被看作《伊莉丝》之后、亚兰德之前的重要过渡点：RPG 化没有退回去，但生活感和角色日常又重新变得更强

## 6. 一句话总结

玛那学园系列最有意思的地方，是它没有把《炼金工房》的 RPG 化继续往“更大战争和更大世界危机”推，而是把这股力量重新装回了学院生活和青春群像里。
