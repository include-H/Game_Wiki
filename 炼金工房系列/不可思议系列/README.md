# 《炼金工房》不可思议系列 Wiki 目录

> 说明：本目录用于整理《炼金工房》在黄昏之后的“不可思议”系列，也就是通常所说的《苏菲》《菲莉丝》《莉迪＆苏瑞》以及后来的《苏菲2》四作。本文主要依据 Atelier 25 周年官方页面、Atelier Wiki、Wikipedia、Koei Tecmo 公开产品页与相关商店页面整理而成，写法以“中文整理型单作 Wiki + 系列索引”为主。

---

## 1. 系列概览

不可思议系列在整个《炼金工房》史上的意义，很大程度上在于它把系列重新拉向了“发现、想象与炼金本身的神奇感”。如果说黄昏系列是在衰退世界里重新强调炼金术的现实价值，那么不可思议系列更像是在问：炼金术除了务实和生存之外，是不是还可以重新变回一种带着童话感和探索欲的东西。

这条线的气质因此非常鲜明。它没有亚兰德那种王都日常的轻快社会感，也没有黄昏那种环境持续变坏的压迫感，而更强调书本、梦境、旅行、绘画、记忆和“原来还有这样的地方 / 这样的配方 / 这样的世界结构”所带来的发现感。

四作内部的定位也很清楚：

- 《苏菲的炼金工房》负责把“不可思议”这条线真正立起来
- 《菲莉丝的炼金工房》把视角真正推向更大的开放旅行舞台
- 《莉迪＆苏瑞的炼金工房》通过双主角和绘画世界，把这一阶段做成较完整的收束
- 《苏菲2》则是多年后补上的正统续作，时间位置夹在《苏菲》与《菲莉丝》之间，既补主角成长，也补整条线的人物感情连接

也正因为如此，不可思议系列最需要先分清的，不只是作品顺序，而是“发售顺序”和“世界观时间顺序”并不完全一致。

## 3. 阅读顺序

如果按发售顺序理解不可思议系列，推荐这样读：

1. 《苏菲的炼金工房 不可思议书的炼金术士》
2. 《菲莉丝的炼金工房 不可思议旅的炼金术士》
3. 《莉迪＆苏瑞的炼金工房 不可思议绘画的炼金术士》
4. 《苏菲的炼金工房2 不可思议梦的炼金术士》

这个顺序最适合看清系列是怎样一步步长出来的，也最符合作品实际推出时的历史脉络，只是要特别注意：《苏菲2》虽然发售最晚，但并不是接在《莉迪＆苏瑞》之后的时间线后续。

如果更重视世界观时间顺序，通常更适合这样读：

1. 《苏菲的炼金工房 不可思议书的炼金术士》
2. 《苏菲的炼金工房2 不可思议梦的炼金术士》
3. 《菲莉丝的炼金工房 不可思议旅的炼金术士》
4. 《莉迪＆苏瑞的炼金工房 不可思议绘画的炼金术士》

对大多数读者来说，两种读法都成立；只是如果你更在意苏菲本人和普拉芙妲相关的情感连贯性，时间顺序会更顺。

## 4. 版本关系与补充说明

不可思议系列最需要先分清的，主要是下面几条关系：

- 《苏菲》《菲莉丝》《莉迪＆苏瑞》三作都存在原版、`DX` 现代整合版关系；今天多数讨论默认都偏向 `DX`
- 这三作在现代平台上还存在《不可思议之炼金术士 三部曲 DX》这种整合入口，因此版本理解相对清楚
- 《苏菲2》不是 `DX` 旧作重整，而是 2022 年推出的全新正统续作
- 《苏菲2》的时间位置夹在初代《苏菲》和《菲莉丝》之间，因此它最容易让人搞混“发售顺序”和“剧情顺序”
- 对《苏菲2》而言，服装和道具类 DLC 不是理解主线的关键；真正重要的是它作为补全角色关系和承前启后的正统新作地位

## 6. 一句话总结

不可思议系列最迷人的地方，是它把《炼金工房》重新写回了“发现新事物本身就值得高兴”的状态。
