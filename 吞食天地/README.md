# 《吞食天地》街机系列 Wiki 目录

> 说明：本目录用于集中整理 Capcom 基于本宫宏志《天地を喰らう》漫画改编的街机游戏条目。本文主要依据 Capcom Arcade Stadium / Capcom Beat 'Em Up Bundle 官方商店页、Wikipedia、MobyGames 与公开资料整理而成，默认包含作品背景、剧情脉络、玩法结构与一定剧透说明。由于街机系《吞食天地》实际只形成了 1989 年《吞食天地》与 1992 年《吞食天地 II 赤壁之战》两部主干作品，因此当前采用“小型系列合并条目”的写法。

---

## 1. 系列概览

很多中文玩家一提到街机《三国志》，脑子里首先浮现的其实不是 KOEI 那条策略《三国志》，也不是后来的《真三国无双》，而是 Capcom 在街机厅里留下的这条《吞食天地》动作线。它最早出发点并不是做严肃历史模拟，而是把《天地を喰らう》这部带有热血漫画色彩的三国改编作品，进一步翻译成骑马冲阵和多人清版。

这一条线实际只有两部街机核心作品：

- 1989 年《吞食天地》，海外常见标题为 `Dynasty Wars`
- 1992 年《吞食天地 II 赤壁之战》，海外常见标题为 `Warriors of Fate`

前者是骑马横卷轴动作，重点在黄巾讨伐与董卓乱政；后者则几乎重做成 Capcom 1990 年代成熟期的清版动作语法，也是中文玩家最熟悉、传播最广的一作。也正因为大众记忆高度集中在二代，本目录虽然按系列合并写，但正文会对《吞食天地 II》投入更多篇幅。

## 3. 阅读方式

如果你主要是想补中文街机厅最常说的“吞食天地 II / 三国志”，直接看主条目里关于《吞食天地 II 赤壁之战》的部分就够了；如果你想知道它为什么会从“骑马冲阵”变成“步战清版”，那就从前作《吞食天地》开始往下读，会更容易看清 Capcom 在 1989 到 1992 年间的设计变化。

---

## 4. 一句话总结

街机《吞食天地》真正有意思的地方，是它用两部作品走完了一次非常典型的 Capcom 进化：先把三国做成马上冲阵，再把它打磨成街机厅里最经典的多人清版动作之一。
