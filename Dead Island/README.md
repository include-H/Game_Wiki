# 《死亡岛》系列 Wiki 目录

> 说明：本目录用于集中整理《死亡岛》（Dead Island）系列相关 Wiki 条目。本文主要依据 Deep Silver 官方页面、Dead Island 官方网站、Steam / PlayStation 商店页、Wikipedia 与 Dead Island Wiki 等公开资料整理而成，默认包含剧情概述、版本关系与一定剧透说明。

---

## 1. 系列概览

《死亡岛》系列最早由 Techland 与 Deep Silver 在 2011 年建立起鲜明轮廓：热带度假天堂、第一人称近战、四人合作、改造武器，以及一种“风景越美，灾难越脏”的强烈反差感。它和很多传统僵尸射击游戏不同，不是把重点放在火力覆盖和资源线管理，而是把肉搏手感、开放区域探索、角色成长和合作推进揉成了一套更偏动作 RPG 的尸潮体验。

如果从系列气质来看，它最有辨识度的地方，其实不是“岛”这个地理概念本身，而是“把末日扔进旅游宣传片背景里”。无论是初代的 Banoi、续作《Riptide》的 Palanai，还是《Dead Island 2》把“岛”转义成被隔离的洛杉矶 HELL-A，系列一直都在写同一种反差：原本属于享乐、消费和度假的空间，被血腥、腐烂和社会崩塌彻底反过来利用。

这个系列的发展大致可以分成三段。第一段是 2011 年初代《Dead Island》奠定基本框架的 Banoi 原点期；第二段是 2013 年《Dead Island: Riptide》延续前作主角团与疫情阴影、但也因“更像强化续篇”而评价分化的过渡期；第三段则是 2023 年《Dead Island 2》在长期开发波折后重新把系列拉回公众视野，用 HELL-A、F.L.E.S.H. 肢解系统和更鲜明的黑色幽默重建品牌形象。

需要额外说明的是，《死亡岛》并不只有编号本体。初代存在《Bloodbath Arena》《Ryder White》这类补充内容，后来又有《Definitive Edition》与《Definitive Collection》；《Riptide》同样有 2016 年的《Definitive Edition》；而《Dead Island 2》则把 `Haus`、`SoLA` 两个剧情扩展与后续免费更新一起并入了完整体验讨论。因此，理解这个系列时，最好把“正传 + 关键扩展 + 重制整合版关系”一起看。

如果把视角再拉宽一点，《死亡岛》和《消逝的光芒》也确实存在很强的开发血缘。前者更像 Techland 在 2010 年代早期把“第一人称近战打僵尸”做出市场辨识度的起点，后者则是在此基础上进一步把跑酷、昼夜高压和城市垂直探索推成独立品牌。也正因为两者关系很近，中文讨论里常会把《消逝的光芒》当成《死亡岛》的精神续作来看；但从正式系列归属、世界观和角色线来看，它们仍应分开整理。

## 2. 分期方式

### 2.1 Banoi 原点期

- [《死亡岛》Wiki](./DEAD_ISLAND_2011_WIKI.md)

这一阶段的核心，是初代一次性把系列最重要的识别元素立住：Banoi 群岛、四位免疫主角、第一人称近战、武器改造、多人合作和“天堂坠毁”的视觉反差。后来的《Definitive Edition》虽然是今天更常见的游玩入口，但系列原点仍然是 2011 年那部作品。

### 2.2 续篇过渡期

- [《死亡岛：激流》Wiki](./DEAD_ISLAND_RIPTIDE_WIKI.md)

这一阶段由《Dead Island: Riptide》构成。它并不是完全推翻前作框架的编号 2，而更像直接承接初代结局的故事续篇：四位幸存者继续前进，新加入 John Morgan，舞台转向 Palanai，环境也从度假区扩展到被洪水与风暴切碎的群岛地带。它在系列中的价值，更多体现在“补完 Banoi 线的后半段”。

### 2.3 HELL-A 复兴期

- [《死亡岛 2》Wiki](./DEAD_ISLAND_2_WIKI.md)

这一阶段的关键，是《Dead Island 2》在多年开发波折后，重新定义了今天玩家对《死亡岛》的印象。它不再延续 Banoi / Palanai 的封闭群岛路线，而是把洛杉矶包装成被隔离的 “HELL-A”，并把系列的黑色幽默、肢解表现和更轻快的动作反馈推到新的主流语境里。

### 2.4 后续未展开内容

目前这一批没有单独展开的相关作品主要包括：

- 《Escape Dead Island》：偏叙事向的外传性质作品，更适合后续单列说明。
- 《Dead Island: Epidemic》：已停运的 MOBA 向衍生作，不属于当前主干阅读路线。
- 《Dead Island: Retro Revenge》：随部分整合版附带的 16-bit 风格横版小游戏，不宜和系列主线正文混写。

## 3. 阅读顺序

如果想按系列主干脉络阅读，比较自然的顺序是：

1. 《Dead Island》
2. 《Dead Island: Riptide》
3. 《Dead Island 2》

如果更在意“完整体验”而不是只看本体，则建议这样理解：

1. 《Dead Island》
2. 《Dead Island》相关关键补充：`Ryder White`、`Bloodbath Arena`
3. 《Dead Island: Riptide》
4. 《Dead Island 2》
5. 《Dead Island 2》剧情扩展：`Haus`、`SoLA`

需要注意的是，《Dead Island 2》在故事时间上与前两作存在明显间隔，气质和结构也比《Riptide》更像一次重振而不是简单顺延。因此，阅读时最好把它看成“同一品牌下的新阶段”，而不是“Riptide 的直接玩法升级版”。

## 5. 版本关系与补充说明

- 2011 年《Dead Island》是系列原点，最早登陆 Windows、PlayStation 3 与 Xbox 360；2014 年推出 OS X 版本，2016 年推出《Dead Island Definitive Edition》并被纳入《Dead Island Definitive Collection》。
- 《Dead Island》最值得一并说明的补充内容是 `Bloodbath Arena` 与 `Ryder White`。前者偏合作生存模式，后者则从 Ryder White 视角补足部分叙事。
- 《Dead Island: Riptide》于 2013 年推出，是接在初代结局之后的直接续篇；2016 年同样推出《Dead Island: Riptide Definitive Edition》，并与初代重制版一起被收进《Definitive Collection》。
- 《Riptide》虽然是系列主干的一部分，但长期以来常被认为更像“强化扩展式续篇”。写作时需要把它的承接价值与创新有限这两点同时说清楚。
- 《Dead Island 2》于 2023 年正式发售，开发过程中历经多次易手，因此它和初代、`Riptide` 在开发谱系上已经不是同一条延续线；最终版本由 Dambuster Studios 完成。
- `Haus` 与 `SoLA` 都不是纯装饰性 DLC，而是《Dead Island 2》最需要并入理解的两段剧情扩展。`Neighborhood Watch` 与 `New Game+` 则属于后续重要免费更新内容。
- 《Dead Island 2 Ultimate Edition》可以视作当前较完整的官方整合入口之一，但讨论作品本身时，仍然应把 2023 年本体首发、两个剧情扩展和后续更新区分开来。
- 如果读者是从《消逝的光芒》回头看《死亡岛》，最值得注意的差别在于：前者更强调跑酷、夜间追逐与城市立体移动，后者则更保留“近战刷装 + 岛屿灾区探索”的原始粗粝感。

---

## 6. 一句话总结

《死亡岛》系列真正让人记住的，不只是“打僵尸很血腥”，而是它总能把本该轻松享乐的空间改造成尸潮地狱，再用近战、合作和黑色幽默把这种反差打得格外具体。
