## 1. 条目概览

《恐龙快打》是 1990 年代 Capcom 街机黄金期最有代表性的清版动作之一。它并没有发展成持续编号的大系列，却靠着极强的题材辨识度，在华语玩家记忆里留下了远超“单部街机作”体量的存在感。凯迪拉克、恐龙、末世城市、黑市猎人、实验怪物和三人合作闯关这些元素，同时出现在一部游戏里，本身就足够怪，也足够好记。

和仓库里很多“系列目录”不同，这个目录当前不是为了拆成多作时间线，而是先围绕 1993 年街机原作建立一篇主条目。原因很简单：《恐龙快打》的历史影响力，主要就集中在这部 Capcom 街机原作本身。

## 3. 阅读方式

如果你想了解这款游戏为什么会在中文玩家记忆里长期占据“街机厅名作”位置，直接从主条目进入就够了。当前这篇主文已经统一整理了：

- 作品基础信息与常见中文译名
- 末世世界观和漫画改编背景
- 八关流程所对应的剧情推进
- 四名主角与主要 Boss 的角色定位
- 玩法结构、枪械系统与凯迪拉克追逐段的特色
- 它在 Capcom 清版动作谱系里的历史位置

## 4. 一句话总结

《恐龙快打》真正厉害的地方，是它把一个听上去像胡闹的题材组合，做成了一部直到今天都还很能打的街机清版动作经典。
