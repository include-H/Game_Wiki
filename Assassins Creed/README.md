# 刺客信条系列 Wiki 目录

> 说明：本目录用于集中整理《刺客信条》系列相关 Wiki 条目。当前内容主要覆盖从《刺客信条 III》到《枭雄》这一段“旧公式后期”主干，以及《自由呐喊》《艾吉奥合集》这类重要合集 / 衍生内容。像《起源》之后的新阶段作品，当前暂不作为本目录的整理重点。

---

## 1. 系列概览

《刺客信条》是 Ubisoft 最具代表性的历史动作冒险系列之一。它最早以“历史题材 + 刺客 / 圣殿骑士秘密战争 + 现代线”建立辨识度，之后逐渐扩展出多个不同阶段：艾吉奥时代的古典城市潜行、北美与加勒比阶段的大空间探索、工业革命时期的城市夺回战，以及更后来的 RPG 化新阶段。

如果按当前目录已经整理的内容来看，可以先大致理解成几条主要脉络：

- 艾吉奥角色归档与补完入口
- 北美独立战争与海盗黄金时代阶段
- 《黑旗》外传与《叛变》这类连接型作品
- 《大革命》《枭雄》所代表的传统城市公式后期阶段

## 3. 推荐阅读顺序

如果是想按当前目录里的历史脉络来阅读，比较适合按下面顺序：

1. 《刺客信条：艾吉奥合集》
2. 《刺客信条 III》
3. 《刺客信条 IV：黑旗》
4. 《刺客信条：自由呐喊》
5. 《刺客信条：叛变》
6. 《刺客信条：大革命》
7. 《刺客信条：枭雄》

如果是想按“系列玩法与气质变化”来读，也可以这样理解：

- 想先看经典角色传奇，就从《艾吉奥合集》进入。
- 想看系列第一次大换代，就读《刺客信条 III》。
- 想看海盗与海战阶段高峰，就读《黑旗》与《自由呐喊》。
- 想看立场反转与时间线补完，就读《叛变》。
- 想看传统城市潜行公式的后期形态，就读《大革命》和《枭雄》。
