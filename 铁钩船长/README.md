> 说明：本目录用于集中整理 1992 年街机游戏《铁钩船长》（Hook）相关 Wiki 条目。本文主要依据 Irem 街机宣传资料、MobyGames、Arcade History、Wikipedia 与公开资料整理而成，默认包含作品背景、剧情概述、角色定位与一定剧透说明。

## 1. 条目概览

《铁钩船长》并不是一个发展出长线编号的系列，但它在街机时代留下的记忆强度并不低。原因很简单：它踩中了几个非常适合街机厅传播的点。第一，它改编自 1991 年斯皮尔伯格电影《Hook》，题材天然有辨识度；第二，它是 Irem 1992 年推出的四人合作清版动作，足够热闹；第三，它把彼得潘、迷失少年、海盗、梦幻岛这些童话元素，翻译成了一部很适合几个人围着一起闯关的街机作品。

当前这个目录不做系列拆分，而是围绕 1992 年街机原作写一篇主条目。因为《铁钩船长》真正值得被记住的核心，也基本集中在这部街机原作本身。

## 3. 阅读方式

如果你想知道这部游戏为什么会在不少玩家心里留下“童话题材街机清版动作”的独特位置，直接进入主条目即可。当前主文已经把电影改编背景、梦幻岛救援主线、可用角色、四人合作结构和它在 Irem 街机作品中的位置统一整理完成。

---

## 4. 一句话总结

《铁钩船长》最特别的地方，是它把一个本来很像家庭奇幻电影的题材，做成了一部既热闹、又带点梦幻危险气味的四人合作街机清版动作。
