# 《英雄传说》系列 Wiki 目录

> 说明：本目录用于集中整理 Falcom 旗下《英雄传说》系列相关 Wiki 条目。本文主要依据日本 Falcom 官方资料、各作公开商店页面、Wikipedia 与相关公开 Wiki 整理而成。当前父目录已经先把早期两作、加卡布三部曲与《轨迹》系列三条主要分支建立起来，后续再继续补充编号关系、支系对照与更多补充页面。

---

## 1. 系列概览

《英雄传说》是 Falcom 旗下历史很长、但内部结构并不适合被简单当成“同一条连续编号正传”的 RPG 系列。它真正麻烦、也真正值得单独建父目录整理的地方，就在于“英雄传说”这个名字下面其实包含了多个阶段、多个支系和多种命名方式。

如果只从今天玩家最熟悉的部分来看，很多人提到《英雄传说》时，第一反应其实已经等同于《轨迹》系列。但从系列史上看，《轨迹》并不是《英雄传说》的全部。在它之前，至少还可以先粗分成两段：

- **早期独立阶段**：`Dragon Slayer: The Legend of Heroes` 与《英雄传说II》
- **加卡布三部曲阶段**：《白发魔女》《朱红的泪》《海之槛歌》

到了《空之轨迹》以后，《英雄传说》才真正进入“轨迹”这条以泽姆利亚大陆为核心的超长篇连载阶段。

也正因为内部结构差异很大，这个父目录更适合像《汤姆·克兰西》那样处理：

- 父目录负责说明《英雄传说》到底包含哪些支系
- 子系列目录分别整理《轨迹》或未来的其他分支
- 单作 Wiki 则仍然放在各自子系列目录内部

换句话说，这里讨论的是“《英雄传说》作为上位系列怎么分”，而不是把所有作品硬塞进同一条主线。

## 3. 除了《轨迹》之外还有什么

如果你只是想先知道“《英雄传说》除了轨迹还有哪些游戏”，当前最简明的答案就是下面这几组：

### 3.1 早期两作

- 《Dragon Slayer: The Legend of Heroes》
- 《Dragon Slayer: The Legend of Heroes II》

这两作属于《英雄传说》最早的起点阶段，当时它和《Dragon Slayer》品牌还有直接承接关系。今天中文语境里提到它们的人已经不算多，但它们确实是《英雄传说》这个系列名的源头。

### 3.2 加卡布三部曲

- 《英雄传说III 白发魔女》
- 《英雄传说IV 朱红的泪》
- 《英雄传说V 海之槛歌》

这三作共享加卡布地区相关设定，因此通常会被单独视作一个完整阶段。很多老玩家提到“前轨时代《英雄传说》”，多数时候说的就是这三部。

### 3.3 轨迹系列

- 《空之轨迹》三部曲
- 《零 / 碧之轨迹》
- 《闪之轨迹》I - IV
- 《创之轨迹》
- 《黎之轨迹》及后续

这部分就是现在最主流、体量也最大的一条《英雄传说》支系。

## 5. 版本关系与补充说明

《英雄传说》父目录最容易混淆的地方，不是某一作的平台差异，而是“哪些作品其实属于不同支系”。

当前需要先分清的几件事包括：

- 《英雄传说》是上位系列名，不等于所有作品共享同一条连续剧情
- 《轨迹》是《英雄传说》之下最庞大、也最主流的一条长篇支系
- 加卡布三部曲与《轨迹》虽然都挂在《英雄传说》名下，但并不是同一世界观连续主线
- 因此目录结构上更合理的做法，是先建《英雄传说》父目录，再让《轨迹》拥有自己的系列 README 与单作页

也就是说，今后如果继续补《英雄传说》这个父目录，核心思路应该始终是：

- 先分支
- 再分期
- 最后才进单作

而不是把“英雄传说”直接当成一条从早期到现在始终不断线的单主线编号系列。
