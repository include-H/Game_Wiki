# 《孤岛余生》系列 Wiki 目录

> 说明：本目录用于集中整理中文语境下常被统称为“孤岛余生”的多条游戏作品线。当前目录保留两种都很常见的对应方式：一条是 G5 / Awem 体系下的 `The Island: Castaway` 与 `The Island: Castaway 2`，另一条则是 Glu Mobile 发行的 Java 手机游戏《Stranded》与《Stranded: Mysteries of Time》。这两条线题材相近，中文名也经常互相套用，但并不是同一系列。本文主要依据 Pocket Gamer、MobyGames、Metacritic、Touch Reviews、Big Fish Games、Shockwave、AppAdvice、G5 官方页面与公开资料整理而成，默认包含剧情概述、版本关系与一定剧透说明。

---

## 1. 系列概览

“孤岛余生”在中文互联网上并不是一个单义标题。很多玩家第一次接触它，是在功能机或早期安卓时代玩到的 Java / 移动端作品；也有人想到的是后来更常见的 G5 `The Island: Castaway` 系列。两者之所以总被混在一起，并不奇怪，因为它们确实共享了一批很容易让人串联起来的表层元素：神秘荒岛、资源管理、采集烹饪、任务推进、轻量求生，以及一种“这岛上肯定还有别的秘密”的持续悬念。

但如果把资料真正拆开看，就会发现这里至少有两条需要分开整理的脉络。

第一条是 Glu Mobile 的《Stranded》系列。它起于 2006 年的 Java 游戏《Stranded》，当时就已经把“海难幸存者 + 岛屿谜团 + 24 小时真实时间 + 捕鱼狩猎种植”的玩法骨架做得很完整；到 2009 年《Stranded: Mysteries of Time》时，系列又把科幻和时间旅行引进来，让舞台从普通荒岛升级到穿越过去、现在与未来的谜题结构，恐龙时代也因此成为很多玩家对“孤岛余生 2”的直观记忆。

第二条是 G5 / Awem 的 `The Island: Castaway` 系列。它更偏休闲冒险和生存管理，最早有 Windows 版本，后来又扩展到 iPhone、iPad、Android、Mac 与 Windows Store。初代围绕海难幸存者 Tom Allison 展开，二代则改成以前传方式补足岛屿原住民历史。对很多后来从 PC 或商店页接触这个名字的玩家来说，这一支往往才是他们印象里的“孤岛余生”。

也就是说，当前目录不是在强行把不相干的东西凑成一个官方大系列，而是承认一个中文使用事实：同一个中文名，现实里对应了两条经常被混称、但确实需要分开核对的作品线。整理时最重要的事情，不是硬说它们“其实完全一样”，而是把各自的正式名称、平台、发售时间、剧情定位和版本关系说清楚。

## 2. 分期方式

### 2.1 Glu Java 原点期

- [《Stranded》Wiki](./STRANDED_2006_WIKI.md)

这一作在当前目录里的独特定位，是“中文玩家记忆里最像老功能机《孤岛余生》原型”的作品。它以海难求生、岛屿谜团和真实时间系统立住了 Glu 这条线的核心气质。

### 2.2 Glu 时空扩展期

- [《Stranded: Mysteries of Time》Wiki](./STRANDED_MYSTERIES_OF_TIME_WIKI.md)

这一作是 Glu 分支最容易被记住的续作。它不再只讲普通荒岛逃生，而是把时空穿越、科研设施、未来危机与恐龙时代都拉进来，成为很多玩家口中的“穿越回恐龙时代的孤岛余生 2”。

### 2.3 G5 海难原点期

- [《孤岛余生》Wiki](./THE_ISLAND_CASTAWAY_WIKI.md)

这一阶段对应 G5 / Awem 分支的初代《The Island: Castaway》。它是海难幸存者 Tom Allison 视角的原点作，也是这条线在 Windows 与后续移动端扩展的开始。

### 2.4 G5 前传扩展期

- [《孤岛余生 2》Wiki](./THE_ISLAND_CASTAWAY_2_WIKI.md)

这一阶段对应《The Island: Castaway 2》。它不是简单接着 Tom 的故事往后写，而是改以前传方式进入原住民部族视角，后来的 `The Island Castaway: Lost World` 也建立在这条线上。

## 3. 阅读顺序

如果你想按“Glu 这条老手机记忆线”阅读，建议顺序是：

1. 《Stranded》
2. 《Stranded: Mysteries of Time》

如果你想按“G5 这条 PC / 后续移动端线”阅读，建议顺序是：

1. 《The Island: Castaway》
2. 《The Island: Castaway 2》

如果你的目的其实是搞清楚“中文名为什么会混”，那比较好的读法反而是：

1. 先读《Stranded》，看最早那批 Java 手机玩家记忆里的“孤岛余生”是什么
2. 再读《Stranded: Mysteries of Time》，理解为什么很多人会记得二代和恐龙、时空穿越绑在一起
3. 然后再看《The Island: Castaway》与《The Island: Castaway 2》，理解为什么另一批玩家会把“孤岛余生”牢牢对应到 G5 / Awem 这组作品

## 4. 当前已覆盖条目

目前这个目录已覆盖 4 篇核心条目，分别来自两条分支：

- Glu 分支：《Stranded》《Stranded: Mysteries of Time》
- G5 分支：《The Island: Castaway》《The Island: Castaway 2》

这 4 篇条目合起来，已经能把“孤岛余生”这个中文名下最容易混淆的几件事说明清楚：

- 为什么很多人说的“Java 版孤岛余生”其实对应的是 Glu 的《Stranded》
- 为什么“二代会穿越到恐龙时代”这件事对应的是《Stranded: Mysteries of Time》
- 为什么另一批玩家又会把“孤岛余生”理解成 G5 的 `The Island: Castaway`
- 为什么这两条线题材相近，却不应该简单当成同一官方系列处理

## 5. 版本关系与补充说明

- Glu 的《Stranded》公开资料最明确能确认的是 2006 年 Java 手机版。当前没有找到足够可靠的官方公开资料，能证明它存在一套独立的官方 PC 版本；因此这里不把它写成 PC 正式双平台作品。
- 《Stranded》与 G5 的《The Island: Castaway》在“荒岛求生 + 任务推进 + 捕鱼狩猎种植”这些表层框架上确实很像，这也是中文名混称的核心原因之一；但从正式英文名、发行商和产品线来看，它们仍是不同作品。
- 《Stranded: Mysteries of Time》是 Glu 分支的续作，MobyGames 可确认其先于 2009 年登陆 J2ME，后续在 2010 年进入 Android 与 iPhone；移动端公开描述也明确提到时间旅行、史前岛屿与恐龙。
- 如果只按玩家印象来记，Glu 二代和 G5 二代很容易被混写，因为它们都带有更大的舞台和更强的奇观感；但前者的关键标签是“时空穿越 / 恐龙”，后者的关键标签是“原住民前传 / Lost World 扩展版”。
- G5 的《The Island: Castaway》与《The Island: Castaway 2》仍然保留在本目录中，因为它们确实也是中文“孤岛余生”最常见的另一组对应。
- 因此，当前目录最稳妥的理解方式不是“单一官方系列”，而是“中文名混称专题目录”。
