## 1. 系列概览

《命令与征服》（Command & Conquer）是即时战略历史上最重要的品牌之一。它不仅帮助奠定了现代 RTS 的主流形态，也在 1990 年代到 2000 年代初构建出多个风格完全不同、却都极具辨识度的平行分支。对很多玩家来说，它不只是一个系列，而几乎是一整个 RTS 黄金时代的代名词。

如果按目前最常见的理解方式来看，《命令与征服》总系列大致可以分成几条主要脉络：

- 泰伯利亚宇宙：从初代《Command & Conquer》到《Tiberian Sun》《Tiberium Wars》《Tiberian Twilight》
- 红色警戒宇宙：从《Red Alert》到《Red Alert 3》，当前已在独立目录中整理
- 将军宇宙：以《Generals》与《Zero Hour》为主，属于相对独立的近未来现实战争分支
- 衍生作品：如第一人称 / 第三人称动作作品《Renegade》等

其中，泰伯利亚线更偏暗色科幻战争史诗；红警线更偏架空历史娱乐战争秀；将军线则走向更贴近现实政治和现代军事冲突的路线。也正是这种“同一品牌下允许完全不同气质宇宙并存”的结构，让《命令与征服》系列长期拥有极强生命力。

## 4. 分支关系说明

《命令与征服》总系列最容易让新读者混淆的地方，就是几个分支并不总是处在同一个世界观里。

大体上可以这样理解：

- 泰伯利亚宇宙是总系列最核心、最完整的连续主线
- 红色警戒最初常被视为泰伯利亚宇宙的一条历史前传分支，但后续作品风格和设定逐步大幅独立
- 将军宇宙基本是相对独立的现实近未来分支
- 《Renegade》属于泰伯利亚宇宙内的动作衍生作

因此，讨论《命令与征服》时，最好先说清自己谈的是：

- 总系列
- 泰伯利亚线
- 红警线
- 将军线

否则很容易把 Kane、GDI、Nod、泰伯利亚矿和尤里、苏联、旭日帝国、GLA 等完全不同系统混在一起。
