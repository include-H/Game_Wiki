## 1. 系列概览

《半条命》（Half-Life）是 Valve 最具代表性的游戏系列之一，也是现代第一人称叙事射击游戏的重要源头。从 1998 年初代《半条命》开始，这个系列逐渐建立起以环境叙事、连续第一人称视角、物理互动、悬念铺陈与高辨识度科幻压迫感为核心的作品气质。

如果按当前目录里的内容划分，这个系列大致可以分成几条主要脉络：

- 初代《半条命》与资料片阶段
- 《半条命 2》与章节续作阶段
- VR 回归作品《半条命：艾利克斯》
- 基于初代重制思路发展出的《黑山》

## 3. 推荐阅读顺序

如果是想按系列时间与作品关系来阅读，比较适合按下面顺序：

1. 《半条命》初代与资料片
2. 《半条命 2》与《第一章》《第二章》《失落的海岸线》
3. 《半条命：艾利克斯》
4. 《黑山》

需要说明的是：

- 《黑山》不是正统时间线上的新作，而是对初代《半条命》的重制解释版本。
- 《半条命：艾利克斯》时间点位于《半条命 2》之前，但其叙事影响会直接触及《第二章》之后的系列未来，因此不建议完全按时间线先读它。
